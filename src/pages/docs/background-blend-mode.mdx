## Основы использования

### Установка режима наложения фона

Используйте утилиты `bg-blend-*` для управления тем, как фоновое изображение(я) элемента должно сливаться с цветом фона.

```html
<div class="bg-blend-multiply ...">
  <!-- ... -->
</div>
```

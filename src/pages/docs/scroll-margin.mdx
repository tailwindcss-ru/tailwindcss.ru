### Использование отрицательных значений

Чтобы использовать отрицательное значение поля прокрутки, поставьте перед именем класса дефис, чтобы преобразовать его в отрицательное значение.

```html
<div class="**-scroll-ml-6** snap-start ...">
  <!-- ... -->
</div>
```

## Использование пользовательских значений

### Настройка вашей темы

По умолчанию для шкалы полей прокрутки Tailwind используется [шкала интервалов по умолчанию](/docs/customizing-spacing). Вы можете настроить масштаб интервалов, отредактировав `theme.spacing` или `theme.extend.spacing` в вашем файле `tailwind.config.js`.

```diff-js {{ filename: 'tailwind.config.js' }}
  module.exports = {
    theme: {
      extend: {
+       spacing: {
+         '96': '24rem',
+       }
      }
    }
  }
```

Кроме того, вы можете настроить только масштаб поля прокрутки, отредактировав `theme.scrollMargin` или `theme.extend.scrollMargin` в вашем файле `tailwind.config.js`.

```diff-js {{ filename: 'tailwind.config.js' }}
  module.exports = {
    theme: {
      extend: {
+       scrollMargin: {
+         '96': '24rem',
+       },
      }
    }
  }
```

Дополнительные сведения о настройке темы по умолчанию смотрите в документации [настройка темы](/docs/theme#customizing-the-default-theme).

## Основы использования

### Установка минимальной высоты

Установите минимальную высоту элемента с помощью утилиты `min-h-*` utilities.

    <div class="absolute w-8 bottom-6">
      <div class="-rotate-90">min-h-80</div>
    </div>
  </div>
  <div class="w-8 bg-blue-500 rounded-lg shadow-lg min-h-64 relative">

  <div class="w-8 bg-blue-500 rounded-lg shadow-lg min-h-40 relative">

## Использование пользовательских значений

### Настройка вашей темы

По умолчанию минимальная шкала высот Tailwind представляет собой комбинацию [шкалы интервалов по умолчанию](/docs/customizing-spacing#default-spacing-scale), а также некоторых дополнительных значений, специфичных для высот.

Вы можете настроить масштаб интервалов, отредактировав `theme.spacing` или `theme.extend.spacing` в файле `tailwind.config.js`.

```diff-js {{ filename: 'tailwind.config.js' }}
  module.exports = {
    theme: {
      extend: {
+       spacing: {
+         '128': '32rem',
+       }
      }
    }
  }
```

Чтобы настроить `min-height` отдельно, используйте раздел `theme.minHeight` вашего файла `tailwind.config.js`.

```diff-js {{ filename: 'tailwind.config.js' }}
  module.exports = {
    theme: {
      extend: {
+       minHeight: {
+         '128': '32rem',
+       }
      }
    }
  }
```
Узнайте больше о настройке темы по умолчанию в документации [настройка темы](/docs/theme#customizing-the-default-theme).

## Основы использования

### Установка минимальной высоты

Установите минимальную высоту элемента с помощью утилит `min-h-0`, `min-h-full` или `min-h-screen`.

```html
<div class="h-48">
  <div class="h-24 **min-h-full**">
    <!-- ... -->
  </div>
</div>
```

## Использование пользовательских значений

### Настройка вашей темы

Вы можете настроить масштаб `min-height`, отредактировав `theme.minHeight` или `theme.extend.minHeight` в вашем файле `tailwind.config.js`.

```diff-js {{ filename: 'tailwind.config.js' }}
  module.exports = {
    theme: {
+     minHeight: {
+       '1/2': '50%',
+     }
    }
  }
```

Дополнительные сведения о настройке темы по умолчанию смотрите в документации [настройка темы](/docs/theme#customizing-the-default-theme).

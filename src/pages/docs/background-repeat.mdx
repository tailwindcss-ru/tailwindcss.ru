## Основы использования

### Повторение

Используйте `bg-repeat` для повторения фонового изображения как по вертикали, так и по горизонтали.

```html {{ example: true }}
<div class="w-full mx-auto max-w-xl bg-white dark:bg-slate-800 dark:ring-1 dark:ring-slate-200/20 h-80 bg-repeat bg-center shadow-xl rounded-lg" style="background-image:url('/img/background-pattern.svg');">
</div>
```

```html
<div class="**bg-repeat** ..." style="background-image: url(...)"></div>
```

### Без повторения

Используйте `bg-no-repeat`, если вы не хотите повторять фоновое изображение.

```html {{ example: true }}
<div class="w-full mx-auto max-w-xl bg-white dark:bg-slate-800 dark:ring-1 dark:ring-slate-200/20 h-64 bg-no-repeat bg-center shadow-xl rounded-lg" style="background-image:url('/img/background-pattern.svg');">
</div>
```

```html
<div class="**bg-no-repeat** bg-center ..." style="background-image: url(...)"></div>
```

### Повторение по горизонтали

Используйте `bg-repeat-x`, чтобы повторять фоновое изображение только по горизонтали.

```html {{ example: true }}
<div class="w-full mx-auto max-w-xl bg-white dark:bg-slate-800 dark:ring-1 dark:ring-slate-200/20 h-64 bg-repeat-x bg-center shadow-xl rounded-lg" style="background-image:url('/img/background-pattern.svg');">
</div>
```

```html
<div class="**bg-repeat-x** bg-center ..." style="background-image: url(...)"></div>
```

### Повторение по вертикали

Используйте `bg-repeat-y`, чтобы повторять фоновое изображение только по вертикали.

```html {{ example: true }}
<div class="w-full mx-auto max-w-xl bg-white dark:bg-slate-800 dark:ring-1 dark:ring-slate-200/20 h-64 bg-repeat-y bg-center shadow-xl rounded-lg" style="background-image:url('/img/background-pattern.svg');">
</div>
```

```html
<div class="**bg-repeat-y** bg-center ..." style="background-image: url(...)"></div>
```

## Основы использования

### Обрезка многострочного текста

Используйте утилиты `line-clamp-*`, чтобы обрезать блок текста после определенного количества строк.

### Снятие зажима линии

Используйте `line-clamp-none`, чтобы отменить ранее примененную утилиту зажима линии.

```html
<p class="line-clamp-3 **lg:line-clamp-none**">
  <!-- ... -->
</p>
```

## Использование пользовательских значений

### Настройка вашей темы

По умолчанию Tailwind предоставляет шесть утилит `line-clamp`. Вы можете настроить эти значения, отредактировав `theme.lineClamp` или `theme.extend.lineClamp` в вашем файле `tailwind.config.js`.

```diff-js tailwind.config.js
  module.exports = {
    theme: {
      extend: {
+       lineClamp: {
+         7: '7',
+       },
      }
    }
  }
```

Узнайте больше о настройке темы по умолчанию в документации [настройка темы](/docs/theme#customizing-the-default-theme) documentation.

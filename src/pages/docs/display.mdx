## Основы использования

### Блочные и Строчные

Используйте утилиты `inline`, `inline-block` и `block` для управления потоком текста и элементов.

```html {{ example: { p: 'none' } }}
<div class="mx-auto max-w-xs bg-white shadow-xl p-4 text-slate-500 text-sm leading-6 sm:text-base sm:leading-7 dark:bg-slate-800 dark:text-slate-400">
  При управлении потоком текста с помощью свойства CSS <span class="inline bg-sky-100 font-bold text-sm text-slate-900 font-mono rounded dark:bg-slate-600 dark:text-slate-200">display: inline</span>
  заставит текст внутри элемента переноситься нормально.
  <br /><br />
  При использовании свойств
  <span class="inline-block bg-sky-100 font-bold text-sm text-slate-900 font-mono rounded dark:bg-slate-600 dark:text-slate-200">display: inline-block</span>
  будет оборачивать элемент, чтобы текст внутри не выходил за пределы его родителя.
  <br /><br />
  Наконец, используя свойство
  <span class="block bg-sky-100 font-bold text-sm text-slate-900 font-mono rounded dark:bg-slate-600 dark:text-slate-200">display: block</span>
  поместит элемент в отдельную строку и заполнит его родительский элемент.
</div>
```

```html
<div>
  При управлении потоком текста с помощью свойства CSS
  <span class="**inline**">display: inline</span>
  вызовет текст внутри элемента для нормального обертывания.

  При использовании свойства <span class="**inline-block**">display: inline-block</span>
  обернет элемент, чтобы предотвратить текст внутри не выходит за пределы своего родителя.

  Наконец, использование свойства <span class="**block**">display: block</span>
  поместит элемент в отдельную строку и заполнит его родительский элемент.
</div>
```

### Корень потока

Используйте утилиту `flow-root` для создания элемента уровня блока с собственным [контекстом форматирования блока](https://developer.mozilla.org/en-US/docs/Web/Guide/CSS/Block_formatting_context).

```html {{ example: { p: 'none' } }}
<div class="mx-auto max-w-xs bg-white shadow-xl p-4 text-slate-500 text-sm leading-6 sm:text-base sm:leading-7 dark:bg-slate-800 dark:text-slate-400">
  <div class="flow-root bg-stripes-purple mb-0.5">
    <div class="my-4 bg-white dark:bg-slate-800">Что ж, позволь мне сказать тебе кое-что, смешной мальчик. Знаешь эту маленькую марку, на которой написано "Публичная библиотека Нью-Йорка"? Ну, это может ничего не значить для вас, но это много значит для меня. Один чертовски много.</div>
  </div>
  <div class="flow-root bg-stripes-purple mt-0.5">
    <div class="my-4 bg-white dark:bg-slate-800">Sure, go ahead, laugh if you want to. I've seen your type before: Flashy, making the scene, flaunting convention. Yeah, I know what you're thinking. What's this guy making such a big stink about old library books? Well, let me give you a hint, junior. </div>
  </div>
</div>
```

```html
<div class="p-4">
  <div class="**flow-root** ...">
    <div class="my-4 ...">Что ж, позволь мне кое-что тебе сказать, ...</div>
  </div>
  <div class="**flow-root** ...">
    <div class="my-4 ...">Конечно, давай, смейся, если хочешь...</div>
  </div>
</div>
```

```html {{ example: { p: 'none' } }}

  <div class="flex items-center gap-4 p-4">

  </div>
</div>
```


### Инлайновый флекс

Используйте утилиту `inline-flex` для создания встроенного гибкого контейнера, в который передается текст.


### Содержимое

Используйте утилиту `contents`, чтобы создать «фантомный» контейнер, дочерние элементы которого действуют как прямые дочерние элементы родителя.

```html {{ example: true }}
<div class="flex gap-4 font-mono text-white text-sm font-bold leading-6 bg-stripes-purple rounded-lg text-center">
  <div class="flex-1 p-4 bg-purple-500 shadow-lg rounded-lg">01</div>
  <div class="contents">
    <div class="flex-1 p-4 bg-purple-500 shadow-lg rounded-lg">02</div>
    <div class="flex-1 p-4 bg-purple-500 shadow-lg rounded-lg">03</div>
  </div>
  <div class="flex-1 p-4 bg-purple-500 shadow-lg rounded-lg">04</div>
</div>
```

```html
<div class="flex ...">
  <div class="flex-1 ...">01</div>
  <div class="contents">
    <div class="flex-1 ...">02</div>
    <div class="flex-1 ...">03</div>
  </div>
  <div class="flex-1 ...">04</div>
</div>
```

### Скрытый

Используйте утилиту `hidden`, чтобы установить для элемента значение `display: none` и удалить его из макета страницы (сравните с `invisible` из документации [visibility](/docs/visibility#invisible)).

```html {{ example: true }}
<div class="flex gap-4 font-mono text-white text-sm font-bold leading-6 bg-stripes-purple rounded-lg text-center">
  <div class="hidden p-4 w-14 h-14 bg-purple-500 shadow-lg rounded-lg">01</div>
  <div class="p-4 w-14 h-14 bg-purple-500 shadow-lg rounded-lg">02</div>
  <div class="p-4 w-14 h-14 bg-purple-500 shadow-lg rounded-lg">03</div>
</div>
```

```html
<div class="flex ...">
  <div class="**hidden** ...">01</div>
  <div>02</div>
  <div>03</div>
</div>
```

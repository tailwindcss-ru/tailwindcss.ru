## Основы использования

### Установка максимальной ширины

Установите максимальную ширину элемента с помощью утилиты `max-w-*`.

```html {{ example: { resizable: true }, }}
<div class="grid gap-4 font-mono font-bold text-xs text-center text-white align-start">
  <div class="grid bg-stripes-blue w-full rounded-lg">
    <div class="px-4 py-2 bg-blue-500 rounded-lg shadow-lg max-w-96">max-w-96</div>
  </div>
  <div class="grid bg-stripes-blue w-full rounded-lg">
    <div class="px-4 py-2 bg-blue-500 rounded-lg shadow-lg max-w-80">max-w-80</div>
  </div>
  <div class="grid bg-stripes-blue w-full rounded-lg">
    <div class="px-4 py-2 bg-blue-500 rounded-lg shadow-lg max-w-64">max-w-64</div>
  </div>
  <div class="grid bg-stripes-blue w-full rounded-lg">
    <div class="px-4 py-2 bg-blue-500 rounded-lg shadow-lg max-w-48">max-w-48</div>
  </div>
  <div class="grid bg-stripes-blue w-full rounded-lg">
    <div class="px-4 py-2 bg-blue-500 rounded-lg shadow-lg max-w-40">max-w-40</div>
  </div>
  <div class="grid bg-stripes-blue w-full rounded-lg">
    <div class="px-4 py-2 bg-blue-500 rounded-lg shadow-lg max-w-32">max-w-32</div>
  </div>
  <div class="grid bg-stripes-blue w-full rounded-lg">
    <div class="px-4 py-2 bg-blue-500 rounded-lg shadow-lg max-w-24">max-w-24</div>
  </div>
</div>
```

```html
<div>
  <div class="w-full **max-w-96** ...">max-w-96</div>
  <div class="w-full **max-w-80** ...">max-w-80</div>
  <div class="w-full **max-w-64** ...">max-w-64</div>
  <div class="w-full **max-w-48** ...">max-w-48</div>
  <div class="w-full **max-w-40** ...">max-w-40</div>
  <div class="w-full **max-w-32** ...">max-w-32</div>
  <div class="w-full **max-w-24** ...">max-w-24</div>
</div>
```

### Размер крупных элементов

Выше `24rem` утилиты `max-w-*` используют именованный масштаб вместо числового, чтобы облегчить угадывание значений.

```html {{ example: { resizable: true } }}
<div class="overflow-hidden relative max-w-md mx-auto bg-white shadow-lg ring-1 ring-black/5 rounded-xl flex items-center gap-6 dark:bg-slate-800 dark:highlight-white/5">
  <img class="absolute -left-6 w-28 h-28 rounded-full shadow-lg" src="https://images.unsplash.com/photo-1501196354995-cbb51c65aaea?ixlib=rb-1.2.1&ixid=MnwxMjA3fDB8MHxwaG90by1wYWdlfHx8fGVufDB8fHx8&auto=format&fit=facearea&facepad=4&w=256&h=256&q=80" />
  <div class="min-w-0 py-5 pl-28 pr-5">
    <div class="text-slate-900 font-medium text-sm sm:text-base truncate dark:text-slate-200">Andrew Alfred</div>
    <div class="text-slate-500 text-sm sm:text-base leading-tight truncate dark:text-slate-400">Assistant to the Traveling Secretary</div>
  </div>
</div>
```

```html
<div class="**max-w-md** ...">
  <!-- ... -->
</div>
```

### Ограничение ваших контрольных точек

Классы `max-w-screen-*` могут использоваться для присвоения элементу максимальной ширины, соответствующей определенной контрольной точке. Эти значения автоматически извлекаются из [раздела `theme.screens`](/docs/theme#screens) вашего файла `tailwind.config.js`.

```html
<div class="max-w-screen-2xl">
  <!-- ... -->
</div>
```

## Использование пользовательских значений

### Настройка вашей темы

По умолчанию максимальный масштаб ширины Tailwind представляет собой комбинацию [масштаба интервалов по умолчанию](/docs/customizing-spacing#default-spacing-scale), а также дополнительного набора именованных размеров для [больших элементов](##sizing-large-elements) исключительно для утилит `max-w-*`.

Вы можете настроить значения глобальной шкалы интервалов, отредактировав `theme.spacing` или `theme.extend.spacing` в файле `tailwind.config.js`.

```diff-js {{ filename: 'tailwind.config.js' }}
  module.exports = {
    theme: {
      extend: {
+       spacing: {
+         '128': '32rem',
+       }
      }
    }
  }
```

Чтобы настроить значения только для утилит `max-w-*`, используйте раздел `theme.maxWidth` вашего файла `tailwind.config.js`.

```diff-js {{ filename: 'tailwind.config.js' }}
  module.exports = {
    theme: {
      extend: {
+       maxWidth: {
+         '8xl': '96rem',
+       }
      }
    }
  }
```

Обратите внимание, что значения, определенные в `theme.maxWidth`, имеют приоритет над значениями, определенными в `theme.spacing`, поэтому добавьте в `theme.spacing` собственное значение, соответствующее одному из именованных размеров по умолчанию (например, `lg` или `xl`) не повлияет на соответствующую утилиту `max-w-*`.

<TipBad>Не переопределяйте именованные размеры в файле `theme.spacing`</TipBad>

```diff-js {{ filename: 'tailwind.config.js' }}
  module.exports = {
    theme: {
      extend: {
+       spacing: {
+         'lg': '30rem',
+       }
      }
    }
  }
```

<TipGood>Используйте `theme.maxWidth` для переопределения именованных размеров</TipGood>

```diff-js {{ filename: 'tailwind.config.js' }}
  module.exports = {
    theme: {
      extend: {
+       maxWidth: {
+         'lg': '30rem',
+       }
      }
    }
  }
```

## Основы использования

### Установка стиля границы

Используйте `border-*` для управления стилем границы элемента.

```html {{ example: true }}
<div class="flex flex-col sm:flex-row items-center justify-around gap-4 text-white text-sm text-center font-bold leading-6">
  <div class="flex flex-col items-center shrink-0">
    <p class="font-medium text-sm text-slate-500 font-mono text-center mb-3 dark:text-slate-400">border-solid</p>
    <button class="px-4 py-2 font-semibold text-sm bg-white text-slate-700 dark:bg-slate-700 dark:text-white rounded-md shadow-sm ring-1 ring-slate-900/5 border-indigo-500 dark:border-sky-500 border-2 border-solid">Кнопка А</button>
  </div>
  <div class="flex flex-col items-center shrink-0">
    <p class="font-medium text-sm text-slate-500 font-mono text-center mb-3 dark:text-slate-400">border-dashed</p>
    <button class="px-4 py-2 font-semibold text-sm bg-white text-slate-700 dark:bg-slate-700 dark:text-white rounded-md shadow-sm ring-1 ring-slate-900/5 border-indigo-500 dark:border-sky-500 border-2 border-dashed">Кнопка А</button>
  </div>
  <div class="flex flex-col items-center shrink-0">
    <p class="font-medium text-sm text-slate-500 font-mono text-center mb-3 dark:text-slate-400">border-dotted</p>
    <button class="px-4 py-2 font-semibold text-sm bg-white text-slate-700 dark:bg-slate-700 dark:text-white rounded-md shadow-sm ring-1 ring-slate-900/5 border-indigo-500 dark:border-sky-500 border-2 border-dotted">Кнопка А</button>
  </div>
  <div class="flex flex-col items-center shrink-0">
    <p class="font-medium text-sm text-slate-500 font-mono text-center mb-3 dark:text-slate-400">border-double</p>
    <button class="px-4 py-2 font-semibold text-sm bg-white text-slate-700 dark:bg-slate-700 dark:text-white rounded-md shadow-sm ring-1 ring-slate-900/5 border-indigo-500 dark:border-sky-500 border-4 border-double">Кнопка А</button>
  </div>
</div>
```

```html
<div class="**border-solid** border-2 border-indigo-600 ..." dark-class="**border-solid** border-2 border-sky-500 ..."></div>
<div class="**border-dashed** border-2 border-indigo-600 ..." dark-class="**border-dashed** border-2 border-sky-500 ..."></div>
<div class="**border-dotted** border-2 border-indigo-600 ..." dark-class="**border-dotted** border-2 border-sky-500 ..."></div>
<div class="**border-double** border-4 border-indigo-600 ..." dark-class="**border-double** border-4 border-sky-500 ..."></div>
```

### Без стиля

Используйте `border-none`, чтобы удалить существующий стиль границы из элемента.

Это чаще всего используется для удаления стиля границы, который был применен к меньшей контрольной точке.

```html {{ example: true }}
<div class="flex justify-center gap-4 text-white text-sm text-center font-bold leading-6">
  <button class="px-4 py-2 font-semibold text-sm bg-white text-slate-700 dark:bg-slate-700 dark:text-white rounded-md shadow-sm ring-1 ring-slate-900/5 dark:ring-white/10 dark:ring-inset border-indigo-500 border-2 border-none">Сохранить изменения</button>
</div>
```

```html
<button class="**border-none** ...">Сохранить изменения</button>
```

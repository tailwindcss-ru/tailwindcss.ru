## Основы использования

### Установка позиции стиля списка

Управляйте положением маркеров и отступом текста в списке с помощью утилит `list-inside` и `list-outside`.

```html {{ example: true }}
<div class="flex flex-col sm:flex-row gap-8">
  <div class="relative">
    <div class="absolute left-8 -top-8 -bottom-8 w-px bg-pink-400/40 dark:bg-rose-400/20"></div>
    <p class="font-medium text-sm text-slate-500 font-mono mb-3 ml-8 dark:text-slate-400">list-inside</p>
    <ul class="list-disc list-inside text-slate-700 bg-white rounded-xl shadow-lg ring-1 ring-slate-900/5 p-4 pl-8 dark:bg-slate-800 dark:ring-0 dark:highlight-white/5 dark:text-slate-400">
      <li>5 стаканов нарезанных белых грибов</li>
      <li>1/2 стакана оливкового масла</li>
      <li>3 фунта сельдерея</li>
    </ul>
  </div>
  <div class="relative">
    <div class="absolute left-8 -top-0 sm:-top-8 -bottom-8 w-px bg-pink-400/40 dark:bg-rose-400/20"></div>
    <p class="font-medium text-sm text-slate-500 font-mono mb-3 ml-8 dark:text-slate-400">list-outside</p>
    <ul class="list-disc list-outside text-slate-700 bg-white rounded-xl shadow-lg ring-1 ring-slate-900/5 p-4 pl-8 dark:bg-slate-800 dark:ring-0 dark:highlight-white/5 dark:text-slate-400">
      <li>5 стаканов нарезанных белых грибов</li>
      <li>1/2 стакана оливкового масла</li>
      <li>3 фунта сельдерея</li>
    </ul>
  </div>
</div>
```

```html
<ul class="list-inside ...">
  <li>5 чашек нарезанных белых грибов</li>
  <!-- ... -->
</ul>

<ul class="list-outside ...">
  <li>5 чашек нарезанных белых грибов</li>
  <!-- ... -->
</ul>
```

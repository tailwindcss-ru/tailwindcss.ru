## Основы использования

### Start

Используйте `justify-items-start` для выравнивания элементов сетки относительно начала их встроенной оси:

```html {{ example: true }}
<div class="grid grid-cols-3 gap-4 font-mono text-white text-sm font-bold leading-6">
  <div class="bg-stripes-sky grid rounded-lg justify-items-start">
    <div class="w-14 h-14 rounded-lg flex items-center justify-center bg-sky-500 shadow-lg">01</div>
  </div>
  <div class="bg-stripes-sky grid rounded-lg justify-items-start">
    <div class="w-14 h-14 rounded-lg flex items-center justify-center bg-sky-500 shadow-lg">02</div>
  </div>
  <div class="bg-stripes-sky grid rounded-lg justify-items-start">
    <div class="w-14 h-14 rounded-lg flex items-center justify-center bg-sky-500 shadow-lg">03</div>
  </div>
  <div class="bg-stripes-sky grid rounded-lg justify-items-start">
    <div class="w-14 h-14 rounded-lg flex items-center justify-center bg-sky-500 shadow-lg">04</div>
  </div>
  <div class="bg-stripes-sky grid rounded-lg justify-items-start">
    <div class="w-14 h-14 rounded-lg flex items-center justify-center bg-sky-500 shadow-lg">05</div>
  </div>
  <div class="bg-stripes-sky grid rounded-lg justify-items-start">
    <div class="w-14 h-14 rounded-lg flex items-center justify-center bg-sky-500 shadow-lg">06</div>
  </div>
</div>
```

```html
<div class="grid **justify-items-start** ...">
  <div>01</div>
  <div>02</div>
  <div>03</div>
  <div>04</div>
  <div>05</div>
  <div>06</div>
</div>
```

### End

Используйте `justify-items-end` для выравнивания элементов сетки относительно конца их встроенной оси:

```html {{ example: true }}
<div class="grid grid-cols-3 gap-4 font-mono text-white text-sm font-bold leading-6">
  <div class="bg-stripes-cyan grid rounded-lg justify-items-end">
    <div class="w-14 h-14 rounded-lg flex items-center justify-center bg-cyan-500 shadow-lg">01</div>
  </div>
  <div class="bg-stripes-cyan grid rounded-lg justify-items-end">
    <div class="w-14 h-14 rounded-lg flex items-center justify-center bg-cyan-500 shadow-lg">02</div>
  </div>
  <div class="bg-stripes-cyan grid rounded-lg justify-items-end">
    <div class="w-14 h-14 rounded-lg flex items-center justify-center bg-cyan-500 shadow-lg">03</div>
  </div>
  <div class="bg-stripes-cyan grid rounded-lg justify-items-end">
    <div class="w-14 h-14 rounded-lg flex items-center justify-center bg-cyan-500 shadow-lg">04</div>
  </div>
  <div class="bg-stripes-cyan grid rounded-lg justify-items-end">
    <div class="w-14 h-14 rounded-lg flex items-center justify-center bg-cyan-500 shadow-lg">05</div>
  </div>
  <div class="bg-stripes-cyan grid rounded-lg justify-items-end">
    <div class="w-14 h-14 rounded-lg flex items-center justify-center bg-cyan-500 shadow-lg">06</div>
  </div>
</div>
```

```html
<div class="grid **justify-items-end** ...">
  <div>01</div>
  <div>02</div>
  <div>03</div>
  <div>04</div>
  <div>05</div>
  <div>06</div>

```

### Center

Используйте `justify-items-center`, чтобы выровнять элементы сетки по их встроенной оси:

```html {{ example: true }}
<div class="grid grid-cols-3 gap-4 font-mono text-white text-sm font-bold leading-6">
  <div class="bg-stripes-fuchsia grid rounded-lg justify-items-center">
    <div class="w-14 h-14 rounded-lg flex items-center justify-center bg-fuchsia-500 shadow-lg">01</div>
  </div>
  <div class="bg-stripes-fuchsia grid rounded-lg justify-items-center">
    <div class="w-14 h-14 rounded-lg flex items-center justify-center bg-fuchsia-500 shadow-lg">02</div>
  </div>
  <div class="bg-stripes-fuchsia grid rounded-lg justify-items-center">
    <div class="w-14 h-14 rounded-lg flex items-center justify-center bg-fuchsia-500 shadow-lg">03</div>
  </div>
  <div class="bg-stripes-fuchsia grid rounded-lg justify-items-center">
    <div class="w-14 h-14 rounded-lg flex items-center justify-center bg-fuchsia-500 shadow-lg">04</div>
  </div>
  <div class="bg-stripes-fuchsia grid rounded-lg justify-items-center">
    <div class="w-14 h-14 rounded-lg flex items-center justify-center bg-fuchsia-500 shadow-lg">05</div>
  </div>
  <div class="bg-stripes-fuchsia grid rounded-lg justify-items-center">
    <div class="w-14 h-14 rounded-lg flex items-center justify-center bg-fuchsia-500 shadow-lg">06</div>
  </div>
</div>
```

```html
<div class="grid **justify-items-center** ...">
  <div>01</div>
  <div>02</div>
  <div>03</div>
  <div>04</div>
  <div>05</div>
  <div>06</div>
</div>
```

### Stretch

Используйте `justify-items-stretch`, чтобы растянуть элементы вдоль их встроенной оси:

```html {{ example: true }}
<div class="grid grid-cols-3 gap-4 justify-items-stretch font-mono text-white text-sm font-bold leading-6">
  <div class="h-14 rounded-lg flex items-center justify-center bg-blue-500 shadow-lg">01</div>
  <div class="h-14 rounded-lg flex items-center justify-center bg-blue-500 shadow-lg">02</div>
  <div class="h-14 rounded-lg flex items-center justify-center bg-blue-500 shadow-lg">03</div>
  <div class="h-14 rounded-lg flex items-center justify-center bg-blue-500 shadow-lg">04</div>
  <div class="h-14 rounded-lg flex items-center justify-center bg-blue-500 shadow-lg">05</div>
  <div class="h-14 rounded-lg flex items-center justify-center bg-blue-500 shadow-lg">06</div>
</div>
```

```html
<div class="grid **justify-items-stretch** ...">
  <div>01</div>
  <div>02</div>
  <div>03</div>
  <div>04</div>
  <div>05</div>
  <div>06</div>
</div>
```

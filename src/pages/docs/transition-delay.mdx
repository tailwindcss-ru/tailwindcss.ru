## Основы использования

### Задержка переходов

Используйте утилиты `delay-*` для управления задержкой перехода элемента.

```html {{ example: { hint: 'Наведите курсор на каждую кнопку, чтобы увидеть ожидаемое поведение' } }}
<div class="flex flex-col sm:flex-row gap-8 sm:gap-0 justify-around text-white text-sm font-bold leading-6">
  <div class="flex flex-col items-center shrink-0">
    <p class="font-medium text-sm text-slate-500 font-mono text-center mb-3 dark:text-slate-400">delay-150</p>
    <button class="px-4 py-2 font-semibold text-sm bg-blue-500 text-white rounded-md shadow-sm hover:scale-125 ease-in-out duration-300 delay-150">Кнопка А</button>
  </div>
  <div class="flex flex-col items-center shrink-0">
    <p class="font-medium text-sm text-slate-500 font-mono text-center mb-3 dark:text-slate-400">delay-300</p>
    <button class="px-4 py-2 font-semibold text-sm bg-blue-500 text-white rounded-md shadow-sm hover:scale-125 ease-in-out duration-300 delay-300">Кнопка Б</button>
  </div>
  <div class="flex flex-col items-center shrink-0">
    <p class="font-medium text-sm text-slate-500 font-mono text-center mb-3 dark:text-slate-400">delay-700</p>
    <button class="px-4 py-2 font-semibold text-sm bg-blue-500 text-white rounded-md shadow-sm hover:scale-125 ease-in-out duration-300 delay-700">Кнопка В</button>
  </div>
</div>
```

```html
<button class="transition **delay-150** duration-300 ease-in-out ...">Кнопка A</button>
<button class="transition **delay-300** duration-300 ease-in-out ...">Кнопка B</button>
<button class="transition **delay-700** duration-300 ease-in-out ...">Кнопка C</button>
```

## Использование пользовательских значений

### Настройка вашей темы

По умолчанию Tailwind предоставляет восемь универсальных утилит задержки перехода. Вы можете настроить эти значения, отредактировав `theme.transitionDelay` или `theme.extend.transitionDelay` в вашем файле `tailwind.config.js`.

```diff-js {{ filename: 'tailwind.config.js' }}
  module.exports = {
    theme: {
      extend: {
+       transitionDelay: {
+         '2000': '2000ms',
+       }
      }
    }
  }
```

Дополнительные сведения о настройке темы по умолчанию смотрите в документации [настройка темы](/docs/theme#customizing-the-default-theme).

## Основы использования

### Auto

Используйте `justify-self-auto` для выравнивания элемента на основе значения свойства сетки `justify-items`:

<Example>
  <div class="grid grid-cols-3 gap-4 justify-items-stretch auto-rows-fr font-mono text-white text-sm font-bold leading-6 text-center">
    <div class="p-4 rounded-lg bg-purple-300 dark:bg-purple-800 dark:text-purple-400">01</div>
    <div class="justify-self-auto p-4 rounded-lg bg-purple-500 shadow-lg">02</div>
    <div class="p-4 rounded-lg bg-purple-300 dark:bg-purple-800 dark:text-purple-400">03</div>
    <div class="p-4 rounded-lg bg-purple-300 dark:bg-purple-800 dark:text-purple-400">04</div>
    <div class="p-4 rounded-lg bg-purple-300 dark:bg-purple-800 dark:text-purple-400">05</div>
    <div class="p-4 rounded-lg bg-purple-300 dark:bg-purple-800 dark:text-purple-400">06</div>
  </div>
</Example>

```html
<div class="grid justify-items-stretch ...">
  <!-- ... -->
  <div class="**justify-self-auto** ...">02</div>
  <!-- ... -->
  <!-- ... -->
  <!-- ... -->
  <!-- ... -->
</div>
```

### Start

Используйте `justify-self-start`, чтобы выровнять элемент сетки относительно начала его встроенной оси:

<Example>
  <div class="grid grid-cols-3 gap-4 justify-items-stretch auto-rows-fr font-mono text-white text-sm font-bold leading-6 text-center">
    <div class="p-4 rounded-lg bg-pink-300 dark:bg-pink-800 dark:text-pink-400">01</div>
    <div class="bg-stripes-pink rounded-lg">
      <div class="justify-self-start p-4 w-14 h-14 rounded-lg bg-pink-500 shadow-lg">02</div>
    </div>
    <div class="p-4 rounded-lg bg-pink-300 dark:bg-pink-800 dark:text-pink-400">03</div>
    <div class="p-4 rounded-lg bg-pink-300 dark:bg-pink-800 dark:text-pink-400">04</div>
    <div class="p-4 rounded-lg bg-pink-300 dark:bg-pink-800 dark:text-pink-400">05</div>
    <div class="p-4 rounded-lg bg-pink-300 dark:bg-pink-800 dark:text-pink-400">06</div>
  </div>
</Example>

```html
<div class="grid justify-items-stretch ...">
  <!-- ... -->
  <div class="**justify-self-start** ...">02</div>
  <!-- ... -->
  <!-- ... -->
  <!-- ... -->
  <!-- ... -->
</div>
```

### Center

Используйте `justify-self-center`, чтобы выровнять элемент сетки по центру его встроенной оси:

<Example>
  <div class="grid grid-cols-3 gap-4 justify-items-stretch auto-rows-fr font-mono text-white text-sm font-bold leading-6 text-center">
    <div class="p-4 rounded-lg bg-indigo-300 dark:bg-indigo-800 dark:text-indigo-400">01</div>
    <div class="justify-self-center grid w-full bg-stripes-indigo rounded-lg">
      <div class="justify-self-center p-4 w-14 h-14 rounded-lg bg-indigo-500 shadow-lg">02</div>
    </div>
    <div class="p-4 rounded-lg bg-indigo-300 dark:bg-indigo-800 dark:text-indigo-400">03</div>
    <div class="p-4 rounded-lg bg-indigo-300 dark:bg-indigo-800 dark:text-indigo-400">04</div>
    <div class="p-4 rounded-lg bg-indigo-300 dark:bg-indigo-800 dark:text-indigo-400">05</div>
    <div class="p-4 rounded-lg bg-indigo-300 dark:bg-indigo-800 dark:text-indigo-400">06</div>
  </div>
</Example>

```html
<div class="grid justify-items-stretch ...">
  <!-- ... -->

  <!-- ... -->
  <!-- ... -->
  <!-- ... -->
  <!-- ... -->
</div>
```

### End

Используйте `justify-self-end`, чтобы выровнять элемент сетки до конца его встроенной оси:

<Example>
  <div class="grid grid-cols-3 gap-4 justify-items-stretch auto-rows-fr font-mono text-white text-sm font-bold leading-6 text-center">
    <div class="p-4 rounded-lg bg-blue-300 dark:bg-blue-800 dark:text-blue-500">01</div>
    <div class="justify-self-end grid w-full bg-stripes-blue rounded-lg">
      <div class="justify-self-end p-4 w-14 h-14 rounded-lg bg-blue-500 shadow-lg">02</div>
    </div>
    <div class="p-4 rounded-lg bg-blue-300 dark:bg-blue-800 dark:text-blue-500">03</div>

    <div class="p-4 rounded-lg bg-blue-300 dark:bg-blue-800 dark:text-blue-500">06</div>
  </div>
</Example>

```html
<div class="grid justify-items-stretch ...">
  <!-- ... -->
  <div class="**justify-self-end** ...">02</div>
  <!-- ... -->
  <!-- ... -->
  <!-- ... -->
  <!-- ... -->
```

### Stretch

Используйте `justify-self-stretch`, чтобы растянуть элемент сетки, чтобы заполнить область сетки на его встроенной оси:

<Example>
  <div class="grid grid-cols-3 gap-4 justify-items-stretch auto-rows-fr font-mono text-white text-sm font-bold leading-6 text-center">
    <div class="bg-stripes-fuchsia rounded-lg">
      <div class="p-4 w-14 h-14 rounded-lg bg-fuchsia-300 dark:bg-fuchsia-800 dark:text-fuchsia-400">01</div>
    </div>
    <div class="justify-self-stretch grid w-full bg-stripes-fuchsia rounded-lg">
      <div class="justify-self-stretch p-4 rounded-lg bg-fuchsia-500 shadow-lg">02</div>
    </div>
    <div class="bg-stripes-fuchsia rounded-lg">
      <div class="p-4 w-14 h-14 rounded-lg bg-fuchsia-300 dark:bg-fuchsia-800 dark:text-fuchsia-400">03</div>
    </div>
    <div class="bg-stripes-fuchsia rounded-lg">
      <div class="p-4 w-14 h-14 rounded-lg bg-fuchsia-300 dark:bg-fuchsia-800 dark:text-fuchsia-400">04</div>
    </div>
    <div class="bg-stripes-fuchsia rounded-lg">
      <div class="p-4 w-14 h-14 rounded-lg bg-fuchsia-300 dark:bg-fuchsia-800 dark:text-fuchsia-400">05</div>
    </div>
    <div class="bg-stripes-fuchsia rounded-lg">
      <div class="p-4 w-14 h-14 rounded-lg bg-fuchsia-300 dark:bg-fuchsia-800 dark:text-fuchsia-400">06</div>
    </div>
  </div>
</Example>

```html
<div class="grid justify-items-start ...">
  <!-- ... -->
  <div class="**justify-self-stretch** ...">02</div>
  <!-- ... -->
  <!-- ... -->
  <!-- ... -->
  <!-- ... -->
</div>
```

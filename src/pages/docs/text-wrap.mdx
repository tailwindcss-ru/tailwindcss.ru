## Основы использования

### Wrap

Используйте `text-wrap`, чтобы перенести переполненный текст на несколько строк в логических точках текста.

```html {{ example: { p: 'none' } }}
<div class="px-4 sm:px-0">
  <div class="grid gap-4 mx-auto max-w-sm bg-white shadow-xl p-8 text-slate-700 dark:bg-slate-800 dark:text-slate-400 text-wrap">
    <h3 class="text-xl dark:text-white font-semibold text-slate-900">Beloved Manhattan soup stand closes<h3>
    <p class="text-sm/6">New Yorkers are facing the winter chill with less warmth this year as the city's most revered soup stand unexpectedly shutters, following a series of events that have left the community puzzled.</p>
  </div>
</div>
```

```html
<article class="**text-wrap** ...">
  <h3>Beloved Manhattan soup stand closes<h3>
  <p>New Yorkers are facing the winter chill...</p>
</article>
```

### No Wrap

Используйте `text-nowrap`, чтобы предотвратить перенос текста и позволить ему переполняться при необходимости.

```html {{ example: { p: 'none' } }}
<div class="grid gap-4 bg-white px-6 py-8 shadow-xl text-slate-700 max-w-sm mx-auto overflow-auto dark:bg-slate-800 dark:text-slate-400 text-nowrap">
  <h3 class="text-xl dark:text-white font-semibold text-slate-900">Beloved Manhattan soup stand closes<h3>
  <p class="text-sm/6">New Yorkers are facing the winter chill with less warmth this year as the city's most revered soup stand unexpectedly shutters, following a series of events that have left the community puzzled.</p>
</div>
```

```html
<article class="**text-nowrap** ...">
  <h3>Beloved Manhattan soup stand closes<h3>
  <p>New Yorkers are facing the winter chill...</p>
</article>
```

### Balance

Используйте `text-balance`, чтобы равномерно распределить текст по каждой строке.

```html {{ example: { p: 'none' } }}
<div class="px-4 sm:px-0">
  <div class="grid gap-4 mx-auto max-w-sm bg-white shadow-xl p-8 text-slate-700 dark:bg-slate-800 dark:text-slate-400">
    <h3 class="text-balance text-xl dark:text-white font-semibold text-slate-900">Beloved Manhattan soup stand closes<h3>
    <p class="text-sm/6">New Yorkers are facing the winter chill with less warmth this year as the city's most revered soup stand unexpectedly shutters, following a series of events that have left the community puzzled.</p>
  </div>
</div>
```

```html
<article>
  <h3 class="**text-balance** ...">Beloved Manhattan soup stand closes<h3>
  <p>New Yorkers are facing the winter chill...</p>
</article>
```

Из соображений производительности браузеры ограничивают балансировку текста блоками длиной ~6 строк или меньше, что делает его лучше всего подходящим для заголовков.

### Pretty

Используйте `text-pretty`, чтобы предотвратить появление висячих строк (одного слова в отдельной строке) в конце текстового блока.

```html {{ example: { p: 'none' } }}
<div class="px-4 sm:px-0">
  <div class="grid gap-4 mx-auto max-w-sm bg-white shadow-xl p-8 text-slate-700 dark:bg-slate-800 dark:text-slate-400 text-pretty">
    <h3 class="text-xl dark:text-white font-semibold text-slate-900">Beloved Manhattan soup stand closes<h3>
    <p class="text-sm/6">New Yorkers are facing the winter chill with less warmth this year as the city's most revered soup stand unexpectedly shutters, following a series of events that have left the community puzzled.</p>
  </div>
</div>
```

```html
<article class="**text-pretty** ...">
  <h3>Beloved Manhattan soup stand closes<h3>
  <p>New Yorkers are facing the winter chill...</p>
</article>
```

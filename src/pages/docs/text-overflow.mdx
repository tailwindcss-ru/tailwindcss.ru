## Основы использования

### Обрезка

Используйте `truncate`, чтобы предотвратить перенос текста, и при необходимости обрезайте переполненный текст с помощью многоточия (<code>…</code>).

```html {{ example: { p: 'none' } }}
<div class="px-4 sm:px-0">
  <div class="mx-auto max-w-sm bg-white shadow-xl p-8 text-slate-700 text-sm leading-6 sm:text-base sm:leading-7 dark:bg-slate-800 dark:text-slate-400">
    <p class="truncate">
      Самое длинное слово в любом из основных словарей английского языка — <span class="text-slate-900 font-medium">пневмоноультрамикроскопическийсиликовулканокониоз,</span>, обозначающее заболевание легких, вызванное вдыханием очень мелких частиц кремнезема. , конкретно из вулкана; с медицинской точки зрения это то же самое, что и силикоз.
    </p>
  </div>
</div>
```

```html
<p class="**truncate** ...">...</p>
```

### Многоточие

Используйте `text-ellipsis`, чтобы обрезать переполняющий текст многоточием (<code>…</code>), если необходимо.

```html {{ example: { p: 'none' } }}
<div class="px-4 sm:px-0">
  <div class="mx-auto max-w-sm bg-white shadow-xl p-8 text-slate-700 text-sm leading-6 sm:text-base sm:leading-7 dark:bg-slate-800 dark:text-slate-400">
    <p class="text-ellipsis overflow-hidden">
      Самое длинное слово в любом из основных словарей английского языка — <span class="text-slate-900 font-medium">пневмоноультрамикроскопическийсиликовулканокониоз,</span>, обозначающее заболевание легких, вызванное вдыханием очень мелких частиц кремнезема. , конкретно из вулкана; с медицинской точки зрения это то же самое, что и силикоз.
    </p>
  </div>
</div>
```

```html
<p class="**text-ellipsis** overflow-hidden ...">...</p>
```

### Зажим переполнения

Используйте `text-clip`, чтобы обрезать текст до предела области содержимого.

```html {{ example: { p: 'none' } }}
<div class="px-4 sm:px-0">
  <div class="mx-auto max-w-sm bg-white shadow-xl p-8 text-slate-700 text-sm leading-6 sm:text-base sm:leading-7 dark:bg-slate-800 dark:text-slate-400">
    <p class="text-clip overflow-hidden">
      Самое длинное слово в любом из основных словарей английского языка — <span class="text-slate-900 font-medium">пневмоноультрамикроскопическийсиликовулканокониоз,</span>, обозначающее заболевание легких, вызванное вдыханием очень мелких частиц кремнезема. , конкретно из вулкана; с медицинской точки зрения это то же самое, что и силикоз.
    </p>
  </div>
</div>
```

```html
<p class="**text-clip** overflow-hidden ...">...</p>
```

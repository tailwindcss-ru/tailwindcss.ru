## Основы использования

### Авто

Используйте `bg-auto` для отображения фонового изображения с размером по умолчанию.

<Example>
  <div class="relative flex items-center justify-center h-48 rounded-lg w-56 sm:w-96 mx-auto overflow-hidden">
    <div class="absolute inset-0 opacity-50 bg-stripes-gray"></div>
    <div class="relative z-10 w-full h-full bg-auto bg-no-repeat bg-center bg-[url(https://images.unsplash.com/photo-1554629947-334ff61d85dc?ixid=MnwxMjA3fDB8MHxwaG90by1wYWdlfHx8fGVufDB8fHx8&ixlib=rb-1.2.1&auto=format&fit=crop&w=128&h=160&q=80)]">
    </div>
  </div>
</Example>

```html
<div class="**bg-auto** bg-no-repeat bg-center ..." style="background-image: url(...)"></div>
```

### Заполнить

Используйте `bg-cover`, чтобы масштабировать фоновое изображение до тех пор, пока оно не заполнит фоновый слой.

<Example>
  <div class="relative flex items-center justify-center rounded-lg overflow-hidden w-56 sm:w-96 mx-auto">
    <div class="absolute inset-0 opacity-50 bg-stripes-gray"></div>
    <div class="relative z-10 w-full h-48 bg-cover bg-no-repeat bg-center bg-[url(https://images.unsplash.com/photo-1554629947-334ff61d85dc?ixid=MnwxMjA3fDB8MHxwaG90by1wYWdlfHx8fGVufDB8fHx8&ixlib=rb-1.2.1&auto=format&fit=crop&w=512&h=640&q=80)]">
    </div>
  </div>
</Example>

```html
<div class="**bg-cover** bg-center ..." style="background-image: url(...)"></div>
```

### Вместить

Используйте `bg-contain`, чтобы масштабировать фоновое изображение до внешних краев без обрезки или растяжения.

<Example>
  <div class="relative flex items-center justify-center rounded-lg overflow-hidden w-56 sm:w-96 mx-auto">
    <div class="absolute inset-0 opacity-50 bg-stripes-gray"></div>
    <div class="relative z-10 w-full h-48 bg-contain bg-no-repeat bg-center sm:bg-top bg-[url(https://images.unsplash.com/photo-1554629947-334ff61d85dc?ixid=MnwxMjA3fDB8MHxwaG90by1wYWdlfHx8fGVufDB8fHx8&ixlib=rb-1.2.1&auto=format&fit=crop&w=512&h=640&q=80)]">
    </div>
  </div>
</Example>

```html
<div class="**bg-contain** bg-center ..." style="background-image: url(...)"></div>
```

## Использование пользовательских значений

### Настройка вашей темы

По умолчанию Tailwind предоставляет утилиты для размеров фона `auto`, `cover` и `contain`. Вы можете изменить, добавить или удалить их, отредактировав раздел `theme.backgroundSize` в вашей конфигурации.

```diff-js tailwind.config.js
  module.exports = {
    theme: {
      backgroundSize: {
        'auto': 'auto',
        'cover': 'cover',
        'contain': 'contain',
+       '50%': '50%',
+       '16': '4rem',
      }
    }
  }
```

Дополнительные сведения о настройке темы по умолчанию смотрите в документации [настройка темы](/docs/theme#customizing-the-default-theme).

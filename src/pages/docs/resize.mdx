## Основы использования

### Изменение размера во всех направлениях

Используйте `resize`, чтобы изменить размер элемента по горизонтали и вертикали.

```html {{ example: { hint: 'Перетащите дескриптор текстовой области в демоверсии, чтобы увидеть ожидаемое поведение' } }}
<div class="w-full flex items-center justify-center">
  <textarea class="resize text-sm p-2 w-80 ring-1 ring-slate-900/10 shadow-sm rounded-md dark:bg-slate-800 dark:ring-0 dark:highlight-white/5" rows="3"></textarea>
</div>
```

```html
<textarea class="**resize** rounded-md"></textarea>
```

### Изменить размер по вертикали

Используйте `resize-y`, чтобы изменить размер элемента по вертикали.

```html {{ example: { hint: 'Перетащите дескриптор текстовой области в демоверсии, чтобы увидеть ожидаемое поведение' } }}
<div class="w-full flex items-center justify-center">
  <textarea class="resize-y text-sm p-2 w-80 ring-1 ring-slate-900/10 shadow-sm rounded-md dark:bg-slate-800 dark:ring-0 dark:highlight-white/5" rows="3"></textarea>
</div>
```

```html
<textarea class="**resize-y** rounded-md"></textarea>
```

### Изменить размер по горизонтали

Используйте `resize-x`, чтобы изменить размер элемента по горизонтали.

```html {{ example: { hint: 'Перетащите дескриптор текстовой области в демоверсии, чтобы увидеть ожидаемое поведение' } }}
<div class="w-full flex items-center justify-center">
  <textarea class="resize-x text-sm p-2 w-80 ring-1 ring-slate-900/10 shadow-sm rounded-md dark:bg-slate-800 dark:ring-0 dark:highlight-white/5" rows="3"></textarea>
</div>
```

```html
<textarea class="**resize-x** rounded-md"></textarea>
```

### Предотвратить изменение размера

Используйте `resize-none`, чтобы предотвратить изменение размера элемента.

```html {{ example: { hint: 'Notice that the textarea handle is gone' } }}
<div class="w-full flex items-center justify-center">
  <textarea class="resize-none text-sm p-2 w-80 ring-1 ring-slate-900/10 shadow-sm rounded-md dark:bg-slate-800 dark:ring-0 dark:highlight-white/5" rows="3"></textarea>
</div>
```

```html
<textarea class="**resize-none** rounded-md"></textarea>
```

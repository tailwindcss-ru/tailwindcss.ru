## Основы использования

### Предотвращение родительской прокрутки

Используйте `overscroll-contain`, чтобы прокрутка в целевой области не запускала прокрутку в родительском элементе, но сохраняла эффекты «отскока» при прокрутке за конец контейнера в операционных системах, которые его поддерживают.

```html {{ example: { p: 'none' } }}
<div class="mx-4 overscroll-contain overflow-auto h-48 bg-white dark:bg-slate-800 shadow-xl p-8 space-y-4 text-slate-700 dark:text-slate-400 text-sm sm:text-base leading-6 sm:leading-7">
  <p>Ну, позволь мне кое-что тебе сказать, смешной мальчик. Знаешь эту маленькую марку, на которой написано "Публичная библиотека Нью-Йорка"? Ну, это может ничего не значить для вас, но это много значит для меня. Чертовски много.</p>
  <p>Конечно, смейтесь, если хотите. Я уже видел ваш тип: кричащий, устраивающий сцену, щеголяющий условностями. Да, я знаю, о чем ты думаешь. Что это за парень поднимает такую вонь из-за старых библиотечных книг? Что ж, позволь мне дать тебе подсказку, младший.</p>
  <p>Возможно, мы сможем жить без библиотек, такие люди, как вы и я. Может быть. Конечно, мы слишком стары, чтобы изменить мир, но как насчет того ребенка, который сел, открыл книгу прямо сейчас в отделении местной библиотеки и нашел рисунки пи-пи и пи-пи на Коте в Шляпа и пять китайских братьев? Разве ОН не заслуживает лучшего?</p>
</div>
```

```html
<div class="**overscroll-contain** ...">Что ж, позволь мне кое-что тебе сказать, ...</div>
```

### Предотвращение отскока при прокрутке

Используйте `overscroll-none`, чтобы прокрутка в целевой области не запускала прокрутку в родительском элементе, а также предотвращала эффекты «отскока» при прокрутке за конец контейнера.

```html {{ example: { p: 'none' } }}
<div class="mx-4 overscroll-none overflow-auto h-48 bg-white dark:bg-slate-800 shadow-xl p-8 space-y-4 text-slate-700 dark:text-slate-400 text-sm sm:text-base leading-6 sm:leading-7">
```

### Использование поведения прокрутки по умолчанию

Используйте `overscroll-auto`, чтобы пользователь мог продолжить прокрутку родительской области прокрутки, когда они достигнут границы основной области прокрутки.

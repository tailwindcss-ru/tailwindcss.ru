## Основы использования

### Установка цвета тени блока

Используйте утилиту `shadow-*`, чтобы изменить цвет существующей тени блока. По умолчанию цветные тени имеют непрозрачность 100%, но вы можете настроить ее с помощью модификатора непрозрачности.

```html {{ example: true }}
<div class="flex flex-col sm:flex-row justify-center gap-8 sm:gap-16">
  <div class="flex flex-col items-center shrink-0">
    <p class="font-medium text-sm text-slate-500 font-mono text-center mb-3 dark:text-slate-400">shadow-cyan-500/50</p>
    <button class="py-2 px-3 bg-cyan-500 text-white text-sm font-semibold rounded-md shadow-lg shadow-cyan-500/50 focus:outline-none">Подписаться</button>
  </div>
  <div class="flex flex-col items-center shrink-0">
    <p class="font-medium text-sm text-slate-500 font-mono text-center mb-3 dark:text-slate-400">shadow-blue-500/50</p>
    <button class="py-2 px-3 bg-blue-500 text-white text-sm font-semibold rounded-md shadow-lg shadow-blue-500/50 focus:outline-none">Подписаться</button>
  </div>
  <div class="flex flex-col items-center shrink-0">
    <p class="font-medium text-sm text-slate-500 font-mono text-center mb-3 dark:text-slate-400">shadow-indigo-500/50</p>
    <button class="py-2 px-3 bg-indigo-500 text-white text-sm font-semibold rounded-md shadow-lg shadow-indigo-500/50 focus:outline-none">Подписаться</button>
  </div>
</div>
```

```html
<button class="bg-cyan-500 **shadow-lg shadow-cyan-500/50** ...">Подписаться</button>
<button class="bg-blue-500 **shadow-lg shadow-blue-500/50** ...">Подписаться</button>
<button class="bg-indigo-500 **shadow-lg shadow-indigo-500/50** ...">Подписаться</button>
```

### Использование теней на цветном фоне

При использовании теней на цветном фоне цветные тени часто могут выглядеть более естественными, чем тени по умолчанию на основе черного, которые, как правило, выглядят серыми и размытыми.

```html {{ example: { p: 'none' } }}
<div class="flex justify-around bg-indigo-300 px-8 pt-10 pb-12 space-x-6">
  <div class="flex flex-col items-center">
    <p class="mb-3 text-sm font-medium text-indigo-900">Default shadow</p>
    <button class="py-2 px-3 bg-white text-indigo-600 text-sm font-semibold rounded-md shadow-lg focus:outline-none">Подписаться</button>
  </div>
  <div class="flex flex-col items-center">
    <p class="mb-3 text-sm font-medium text-indigo-900">Colored shadow</p>
    <button class="py-2 px-3 bg-white text-indigo-600 text-sm font-semibold rounded-md shadow-lg shadow-indigo-500/40 focus:outline-none">Подписаться</button>
  </div>
</div>
```

```html
<!-- Default shadow-->
<button class="shadow-lg">Подписаться</button>

<!-- Colored shadow-->
<button class="shadow-lg **shadow-indigo-500/40** ...">Подписаться</button>
```

### Сохранение цвета тени при изменении размера тени

Обратите внимание, что если вы измените размер [box shadow](/docs/box-shadow) в другой контрольной точке или в другом состоянии, например при наведении курсора, он переопределит все существующие цвета тени.

```html
<!-- This shadow will NOT be red at medium screens and above -->
<div class="shadow-lg shadow-red-500 **md:shadow-xl**"></div>
```

Чтобы сохранить цвет тени в этих состояниях, его необходимо переопределить:

```html
<!-- This shadow WILL be red at medium screens and above -->
<div class="shadow-lg shadow-red-500 **md:shadow-xl md:shadow-red-500**"></div>
```

Это необходимо, чтобы Tailwind работал таким образом, иначе не было бы возможности вернуться к тени по умолчанию.

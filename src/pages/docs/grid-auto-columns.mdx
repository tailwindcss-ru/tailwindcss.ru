## Основы использования

### Определение размеров неявно созданных столбцов сетки

Используйте утилиты `auto-cols-*` для управления размером неявно созданных столбцов сетки.

```html
<div class="grid grid-flow-col **auto-cols-max**">
  <div>01</div>
  <div>02</div>
  <div>03</div>
</div>
```

## Использование пользовательских значений

### Настройка вашей темы

По умолчанию Tailwind включает четыре универсальных `grid-auto-columns`. Вы можете настроить эти значения, отредактировав `theme.gridAutoColumns` или `theme.extend.gridAutoColumns` в вашем файле `tailwind.config.js`.

```diff-js {{ filename: 'tailwind.config.js' }}
  module.exports = {
    theme: {
      extend: {
+       gridAutoColumns: {
+         '2fr': 'minmax(0, 2fr)',
+       }
      }
    }
  }
```

Дополнительные сведения о настройке темы по умолчанию смотрите в документации [настройка темы](/docs/theme#customizing-the-default-theme).

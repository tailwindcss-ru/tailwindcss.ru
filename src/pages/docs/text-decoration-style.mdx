## Основы использования

### Настройка стиля оформления текста

Используйте утилиты `decoration-*`, чтобы изменить стиль [оформления текста элемента](/docs/text-decoration).

```html {{ example: true }}
<div class="flex flex-col gap-8 text-slate-900 dark:text-slate-200">
  <div>
    <div class="font-medium text-sm text-slate-500 font-mono mb-3 dark:text-slate-400">decoration-solid</div>
    <p class="text-lg font-medium underline decoration-solid">Быстрая коричневая лиса прыгает через ленивую собаку.</p>
  </div>
  <div>
    <div class="font-medium text-sm text-slate-500 font-mono mb-3 dark:text-slate-400">decoration-double</div>
    <p class="text-lg font-medium underline decoration-double">Быстрая коричневая лиса прыгает через ленивую собаку.</p>
  </div>
  <div>
    <div class="font-medium text-sm text-slate-500 font-mono mb-3 dark:text-slate-400">decoration-dotted</div>
    <p class="text-lg font-medium underline decoration-dotted">Быстрая коричневая лиса прыгает через ленивую собаку.</p>
  </div>
  <div>
    <div class="font-medium text-sm text-slate-500 font-mono mb-3 dark:text-slate-400">decoration-dashed</div>
    <p class="text-lg font-medium underline decoration-dashed">Быстрая коричневая лиса прыгает через ленивую собаку.</p>
  </div>
  <div>
    <div class="font-medium text-sm text-slate-500 font-mono mb-3 dark:text-slate-400">decoration-wavy</div>
    <p class="text-lg font-medium underline decoration-wavy">Быстрая коричневая лиса прыгает через ленивую собаку.</p>
  </div>
</div>
```

```html
<p class="underline **decoration-solid** ...">Быстрая коричневая лиса...</p>
<p class="underline **decoration-double** ...">Быстрая коричневая лиса...</p>
<p class="underline **decoration-dotted** ...">Быстрая коричневая лиса...</p>
<p class="underline **decoration-dashed** ...">Быстрая коричневая лиса...</p>
<p class="underline **decoration-wavy** ...">Быстрая коричневая лиса...</p>
```

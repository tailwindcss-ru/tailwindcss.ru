## Основы использования

### Установка соотношения сторон

Используйте утилиту `aspect-*`, чтобы установить желаемое соотношение сторон элемента.

```html {{ example: { resizable: true } }}
<iframe class="w-full aspect-video rounded-lg shadow-lg" src="https://www.youtube.com/embed/dQw4w9WgXcQ" frameborder="0" allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture" allowfullscreen></iframe>
```

```html
<iframe class="w-full **aspect-video** ..." src="https://www.youtube.com/..."></iframe>
```

Tailwind не включает в себя большой набор значений соотношения сторон по умолчанию, так как проще использовать произвольные значения. См. Раздел [произвольные значения](#arbitrary-values) для получения дополнительной информации.

### Поддержка браузера

Утилиты `aspect-*` используют собственное CSS-свойство `aspect-ratio`, которое не поддерживалось в Safari до версии 15. До популяризации Safari 15 [aspect-ratio](https://github.com/tailwindlabs/tailwindcss-aspect-ratio) — хорошая альтернатива.

## Использование пользовательских значений

### Настройка вашей темы

По умолчанию Tailwind предоставляет минимальный набор `aspect-ratio`. Вы можете настроить эти значения, отредактировав `theme.aspectRatio` или `theme.extend.aspectRatio` в вашем файле `tailwind.config.js`.

```diff-js {{ filename: 'tailwind.config.js' }}
  module.exports = {
    theme: {
      extend: {
        aspectRatio: {
+         '4/3': '4 / 3',
        },
      }
    }
  }
```

Дополнительные сведения о настройке темы по умолчанию смотрите в документации [настройка темы](/docs/theme#customizing-the-default-theme).

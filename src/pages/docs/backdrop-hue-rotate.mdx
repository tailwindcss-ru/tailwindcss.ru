```html
<div class="**backdrop-hue-rotate-90** bg-white/30 ...">
  <!-- ... -->
</div>
<div class="**backdrop-hue-rotate-180** bg-white/30 ...">
  <!-- ... -->
</div>
<div class="**-backdrop-hue-rotate-60** bg-white/30 ...">
  <!-- ... -->

```

### Использование отрицательных значений

Чтобы использовать отрицательное значение поворота оттенка фона, поставьте перед именем класса дефис, чтобы преобразовать его в отрицательное значение.

```html
<div class="**-backdrop-hue-rotate-60**">
  <!-- ... -->
</div>
```

## Использование пользовательских значений

### Настройка вашей темы

По умолчанию Tailwind включает в себя несколько утилит общего назначения `backdrop-hue-rotate`. Вы можете настроить эти значения, отредактировав `theme.backdropHueRotate` или `theme.extend.backdropHueRotate` в вашем файле `tailwind.config.js`.

```diff-js {{ filename: 'tailwind.config.js' }}
  module.exports = {
    theme: {
      extend: {
+       backdropHueRotate: {
+         '-270': '-270deg',
+         270: '270deg',
+       }
      }
    }
  }
```

Дополнительные сведения о настройке темы по умолчанию смотрите в документации [настройка темы](/docs/theme#customizing-the-default-theme).

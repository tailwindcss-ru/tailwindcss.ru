## Основы использования

### Отключение выделения текста

Используйте `select-none`, чтобы предотвратить выделение текста в элементе и его дочерних элементах.

<Example hint="Try selecting the text to see the expected behaviour">
  <div class="flex justify-center">
    <div class="select-none inline-flex text-center bg-white text-slate-900 font-semibold font-sans text-sm shadow-lg rounded-lg ring-1 ring-slate-900/5 px-4 py-3 dark:bg-slate-800 dark:ring-0 dark:highlight-white/5 dark:text-slate-200">
      Быстрая коричневая лиса прыгает через ленивую собаку.
    </div>
  </div>
</Example>

```html
<div class="**select-none** ...">
  Быстрая коричневая лиса прыгает через ленивую собаку.
</div>
```

### Разрешение выделения текста

Используйте `select-text`, чтобы разрешить выделение текста в элементе и его дочерних элементах.

      Быстрая коричневая лиса прыгает через ленивую собаку.

  </div>

```html
<div class="**select-text** ...">

```

### Выделение всего текста одним щелчком мыши

Используйте `select-all`, чтобы автоматически выделять весь текст в элементе, когда пользователь щелкает мышью.

<Example hint="Try selecting the text to see the expected behaviour">
  <div class="flex justify-center">
    <div class="select-all inline-flex text-center bg-white text-slate-900 font-semibold font-sans text-sm shadow-lg rounded-lg ring-1 ring-slate-900/5 px-4 py-3 dark:bg-slate-800 dark:ring-0 dark:highlight-white/5 dark:text-slate-200">
      Быстрая коричневая лиса прыгает через ленивую собаку.
    </div>
  </div>
</Example>

```html
<div class="**select-all** ...">
  Быстрая коричневая лиса прыгает через ленивую собаку.
</div>
```

### Использование автоматического выбора поведения

Используйте `select-auto`, чтобы использовать поведение браузера по умолчанию для выбора текста. Полезно для отмены других классов, таких как `select-none`, в разных контрольных точках.

<Example hint="Попробуйте выделить текст, чтобы увидеть ожидаемое поведение">

## Основы использования

### Установка толщины шрифта

Управляйте толщиной шрифта элемента с помощью утилит `font-{weight}`.

```html {{ example: true }}
<div class="flex flex-col gap-8">
  <div>
    <span class="font-medium text-sm text-slate-500 font-mono mb-3 dark:text-slate-400">font-light</span>
    <p class="font-light text-lg text-slate-900 dark:text-slate-200">
      Быстрая коричневая лиса прыгает через ленивую собаку.
    </p>
  </div>
  <div>
    <span class="font-medium text-sm text-slate-500 font-mono mb-3 dark:text-slate-400">font-normal</span>
    <p class="font-normal text-lg text-slate-900 dark:text-slate-200">
      Быстрая коричневая лиса прыгает через ленивую собаку.
    </p>
  </div>
  <div>
    <span class="font-medium text-sm text-slate-500 font-mono mb-3 dark:text-slate-400">font-medium</span>
    <p class="font-medium text-lg text-slate-900 dark:text-slate-200">
      Быстрая коричневая лиса прыгает через ленивую собаку.
    </p>
  </div>
  <div>
    <span class="font-medium text-sm text-slate-500 font-mono mb-3 dark:text-slate-400">font-semibold</span>
    <p class="font-semibold text-lg text-slate-900 dark:text-slate-200">
      Быстрая коричневая лиса прыгает через ленивую собаку.
    </p>
  </div>
  <div>
    <span class="font-medium text-sm text-slate-500 font-mono mb-3 dark:text-slate-400">font-bold</span>
    <p class="font-bold text-lg text-slate-900 dark:text-slate-200">
      Быстрая коричневая лиса прыгает через ленивую собаку.
    </p>
  </div>
</div>
```

```html
<p class="**font-light** ...">Быстрая коричневая лиса ...</p>
<p class="**font-normal** ...">Быстрая коричневая лиса ...</p>
<p class="**font-medium** ...">Быстрая коричневая лиса ...</p>
<p class="**font-semibold** ...">Быстрая коричневая лиса ...</p>
<p class="**font-bold** ...">Быстрая коричневая лиса ...</p>
```

## Использование пользовательских значений

### Настройка вашей темы

По умолчанию Tailwind предоставляет девять утилит `font-weight`. Вы можете изменить, добавить или удалить их, отредактировав раздел `theme.fontWeight` вашей конфигурации Tailwind.

```diff-js {{ filename: 'tailwind.config.js' }}
  module.exports = {
    theme: {
      fontWeight: {
-       thin: '100',
+       hairline: '100',
        extralight: '200',
        light: '300',
        normal: '400',
        medium: '500',
        semibold: '600',
        bold: '700',
-       extrabold: '800',
+       'extra-bold': '800',
        black: '900',
      }
    }
  }
```

Дополнительные сведения о настройке темы по умолчанию смотрите в документации [настройка темы](/docs/theme#customizing-the-default-theme).

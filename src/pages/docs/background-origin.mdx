## Основы использования

### Установка источника фона

Используйте `bg-origin-border`, `bg-origin-padding` и `bg-origin-content` для управления отображением фона элемента.

```html {{ example: { p: 'none' } }}
<div class="flex sm:block overflow-scroll sm:overflow-visible scroll-p-8">
<div class="flex items-center justify-around gap-12 sm:gap-4 font-mono font-bold shrink-0 p-8">
  <div class="flex flex-col items-center shrink-0">
    <p class="font-medium text-sm text-slate-500 font-mono text-center mb-3 dark:text-slate-400">bg-origin-border</p>
    <div class="bg-origin-border bg-cover p-3 border-4 border-white/75 dark:border-white/40 border-dashed bg-slate-300 bg-no-repeat w-24 h-24 object-cover rounded-lg shadow-xl bg-[url(https://images.unsplash.com/photo-1554629947-334ff61d85dc?ixid=MnwxMjA3fDB8MHxwaG90by1wYWdlfHx8fGVufDB8fHx8&ixlib=rb-1.2.1&auto=format&fit=crop&w=320&h=320&q=80)]"></div>
  </div>
  <div class="flex flex-col items-center shrink-0">
    <p class="font-medium text-sm text-slate-500 font-mono text-center mb-3 dark:text-slate-400">bg-origin-padding</p>
    <div class="bg-origin-padding bg-cover p-3 border-4 border-white/75 dark:border-white/25 border-dashed bg-slate-300 dark:bg-slate-700 bg-no-repeat w-24 h-24 object-cover rounded-lg shadow-xl bg-[url(https://images.unsplash.com/photo-1554629947-334ff61d85dc?ixid=MnwxMjA3fDB8MHxwaG90by1wYWdlfHx8fGVufDB8fHx8&ixlib=rb-1.2.1&auto=format&fit=crop&w=320&h=320&q=80)]"></div>
  </div>
  <div class="flex flex-col items-center shrink-0">
    <p class="font-medium text-sm text-slate-500 font-mono text-center mb-3 dark:text-slate-400">bg-origin-content</p>
    <div class="bg-origin-content bg-cover p-3 border-4 border-white/75 dark:border-white/25 border-dashed bg-slate-300 dark:bg-slate-700 bg-no-repeat w-24 h-24 object-cover rounded-lg shadow-xl bg-[url(https://images.unsplash.com/photo-1554629947-334ff61d85dc?ixid=MnwxMjA3fDB8MHxwaG90by1wYWdlfHx8fGVufDB8fHx8&ixlib=rb-1.2.1&auto=format&fit=crop&w=320&h=320&q=80)]"></div>
  </div>
</div>
</div>
```

```html
<div class="**bg-origin-border** p-4 border-4 border-dashed ..." style="background-image: url(...)"></div>
<div class="**bg-origin-padding** p-4 border-4 border-dashed ..." style="background-image: url(...)"></div>
<div class="**bg-origin-content** p-4 border-4 border-dashed ..." style="background-image: url(...)"></div>
```

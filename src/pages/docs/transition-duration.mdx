## Основы использования

### Изменение продолжительности перехода

Используйте утилиты `duration-*` для управления длительностью перехода элемента.

```html {{ example: { hint: 'Наведите курсор на каждую кнопку, чтобы увидеть ожидаемое поведение' } }}
<div class="flex flex-col sm:flex-row gap-8 sm:gap-0 justify-around text-white text-sm font-bold leading-6">
  <div class="flex flex-col items-center shrink-0">
    <p class="font-medium text-sm text-slate-500 font-mono text-center mb-3 dark:text-slate-400">duration-150</p>
    <button class="px-4 py-2 font-semibold text-sm bg-violet-500 text-white rounded-md shadow-sm hover:scale-125 ease-in-out duration-150">Кнопка А</button>
  </div>
  <div class="flex flex-col items-center shrink-0">
    <p class="font-medium text-sm text-slate-500 font-mono text-center mb-3 dark:text-slate-400">duration-300</p>
    <button class="px-4 py-2 font-semibold text-sm bg-violet-500 text-white rounded-md shadow-sm hover:scale-125 ease-in-out duration-300">Кнопка Б</button>
  </div>
  <div class="flex flex-col items-center shrink-0">
    <p class="font-medium text-sm text-slate-500 font-mono text-center mb-3 dark:text-slate-400">duration-700</p>
    <button class="px-4 py-2 font-semibold text-sm bg-violet-500 text-white rounded-md shadow-sm hover:scale-125 ease-in-out duration-700">Кнопка В</button>
  </div>
</div>
```

```html
<button class="transition **duration-150** ease-in-out ...">Кнопка A</button>
<button class="transition **duration-300** ease-in-out ...">Кнопка B</button>
<button class="transition **duration-700** ease-in-out ...">Кнопка C</button>
```

## Использование пользовательских значений

### Настройка вашей темы

По умолчанию Tailwind предоставляет восемь универсальных утилит для продолжительности перехода. Вы можете настроить эти значения, отредактировав `theme.transitionDuration` или `theme.extend.transitionDuration` в вашем файле `tailwind.config.js`.

```diff-js {{ filename: 'tailwind.config.js' }}
  module.exports = {
    theme: {
      extend: {
+       transitionDuration: {
+         '2000': '2000ms',
+       }
      }
    }
  }
```

Дополнительные сведения о настройке темы по умолчанию смотрите в документации [настройка темы](/docs/theme#customizing-the-default-theme).

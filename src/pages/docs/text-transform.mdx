## Основы использования

### Преобразование текста

`uppercase` и `lowercase` преобразуют текст в верхний и нижний регистр соответственно, тогда как утилита `capitalize` преобразует текст в регистр заголовка. Утилиту `normal-case` можно использовать для сохранения исходного регистра - обычно для сброса заглавных букв в разных контрольных точках.

<Example>
  <div class="flex flex-col gap-8">
    <div>
      <span class="font-medium text-sm text-gray-500 font-mono mb-3">normal-case</span>
      <p class="normal-case text-lg font-medium text-gray-900">
        Быстрая коричневая лиса прыгает через ленивую собаку.
      </p>
    </div>
    <div>
      <span class="font-medium text-sm text-gray-500 font-mono mb-3">uppercase</span>
      <p class="uppercase text-lg font-medium text-gray-900">
        Быстрая коричневая лиса прыгает через ленивую собаку.
      </p>
    </div>
    <div>
      <span class="font-medium text-sm text-gray-500 font-mono mb-3">lowercase</span>
      <p class="lowercase text-lg font-medium text-gray-900">
        Быстрая коричневая лиса прыгает через ленивую собаку.
      </p>
    </div>
    <div>
      <span class="font-medium text-sm text-gray-500 font-mono mb-3">capitalize</span>
      <p class="capitalize text-lg font-medium text-gray-900">
        Быстрая коричневая лиса прыгает через ленивую собаку.
      </p>
    </div>
  </div>
</Example>

```html
<p class="**normal-case** ...">Быстрая коричневая лиса ...</p>
<p class="**uppercase** ...">Быстрая коричневая лиса ...</p>
<p class="**lowercase** ...">Быстрая коричневая лиса ...</p>
<p class="**capitalize** ...">Быстрая коричневая лиса ...</p>
```

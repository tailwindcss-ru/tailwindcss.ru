## Основы использования

### Создание элемента в оттенках серого

Используйте утилиты `backdrop-grayscale` и `backdrop-grayscale-0` для управления отображением фона элемента в оттенках серого или в полноцветном.

    <div class="relative">

## Использование пользовательских значений

### Настройка вашей темы

По умолчанию Tailwind включает в себя несколько утилит общего назначения `backdrop-grayscale`. Вы можете настроить эти значения, отредактировав `theme.backdropGrayscale` или `theme.extend.backdropGrayscale` в вашем файле `tailwind.config.js`.

```diff-js {{ filename: 'tailwind.config.js' }}
  module.exports = {
    theme: {
      extend: {
+       backdropGrayscale: {
+         50: '.5',
+       }
      }
    }
  }
```

Дополнительные сведения о настройке темы по умолчанию смотрите в документации [настройка темы](/docs/theme#customizing-the-default-theme).

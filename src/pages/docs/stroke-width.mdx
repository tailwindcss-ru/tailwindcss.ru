## Основы использования

### Установка ширины штриха

Используйте утилиты `stroke-{width}`, чтобы установить ширину штриха SVG.

<Example>
  <div class="flex items-center justify-center space-x-8">
    <svg class="stroke-indigo-500 stroke-1" width="48" height="48" viewBox="0 0 48 48" fill="none" xmlns="http://www.w3.org/2000/svg"> <circle cx="24" cy="24" r="23" stroke-linejoin="round"/> <path d="M23 1C23 1 15 10.4901 15 24C15 37.5099 23 47 23 47" stroke-linejoin="round"/> <path d="M25 1C25 1 33 10.4901 33 24C33 37.5099 25 47 25 47" stroke-linejoin="round"/> <path d="M1 24H47"/> </svg>
    <svg class="stroke-indigo-500 stroke-2" width="48" height="48" viewBox="0 0 48 48" fill="none" xmlns="http://www.w3.org/2000/svg"> <circle cx="24" cy="24" r="23" stroke-linejoin="round"/> <path d="M23 1C23 1 15 10.4901 15 24C15 37.5099 23 47 23 47" stroke-linejoin="round"/> <path d="M25 1C25 1 33 10.4901 33 24C33 37.5099 25 47 25 47" stroke-linejoin="round"/> <path d="M1 24H47"/> </svg>
  </div>
</Example>

```html
<svg class="**stroke-1** ..."></svg>
<svg class="**stroke-2** ..."></svg>
```

Это может быть полезно для стилизации наборов значков, таких как [Heroicons](https://heroicons.com).

## Использование пользовательских значений

### Настройка вашей темы

По умолчанию Tailwind включает в себя утилиты `stroke-width` для создания базовых сеток, содержащих до 6 строк одинаковой ширины. Вы можете настроить эти значения, отредактировав `theme.strokeWidth` или `theme.extend.strokeWidth` в вашем файле `tailwind.config.js`.

Здесь у вас есть прямой доступ к свойству CSS `stroke-width`, так что вы можете сделать свои пользовательские значения строк как общими, так и сложными и специфичными для сайта, как вам нравится.

```diff-js tailwind.config.js
  module.exports = {
    theme: {
      extend: {
+       strokeWidth: {
+         '2': '2px',
+       }
      }
    }
  }
```

Дополнительные сведения о настройке темы по умолчанию смотрите в документации [настройка темы](/docs/theme#customizing-the-default-theme).

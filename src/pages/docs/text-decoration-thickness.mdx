## Основы использования

### Настройка толщины оформления текста

Используйте утилиту `decoration-*`, чтобы изменить толщину [текстового оформления элемента](/docs/text-decoration).

```html {{ example: true }}
<div class="flex flex-col gap-8 text-slate-900 dark:text-slate-200">
  <div>
    <div class="font-medium text-sm text-slate-500 font-mono mb-3 dark:text-slate-400">decoration-1</div>
    <p class="text-lg font-medium underline decoration-1">Быстрая коричневая лиса прыгает через ленивую собаку.</p>
  </div>
  <div>
    <div class="font-medium text-sm text-slate-500 font-mono mb-3 dark:text-slate-400">decoration-2</div>
    <p class="text-lg font-medium underline decoration-2">Быстрая коричневая лиса прыгает через ленивую собаку.</p>
  </div>
  <div>
    <div class="font-medium text-sm text-slate-500 font-mono mb-3 dark:text-slate-400">decoration-4</div>
    <p class="text-lg font-medium underline decoration-4">Быстрая коричневая лиса прыгает через ленивую собаку.</p>
  </div>
</div>
```

```html
<p class="underline **decoration-1** ...">Быстрая коричневая лиса...</p>
<p class="underline **decoration-2** ...">Быстрая коричневая лиса...</p>
<p class="underline **decoration-4** ...">Быстрая коричневая лиса...</p>
```

## Использование пользовательских значений

### Настройка вашей темы

Вы можете настроить утилиты `decoration-*`, отредактировав `theme.textDecorationThickness` или `theme.extend.textDecorationThickness` в файле `tailwind.config.js`.

```diff-js {{ filename: 'tailwind.config.js' }}
  module.exports = {
    theme: {
      extend: {
+       textDecorationThickness: {
+         3: '3px',
+       }
      }
    }
  }
```

Дополнительные сведения о настройке темы по умолчанию смотрите в документации [настройка темы](/docs/theme#customizing-the-default-theme).

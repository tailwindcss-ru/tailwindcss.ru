## Основы использования

### Размер неявно созданных строк сетки

Используйте утилиты `auto-rows-{size}` для управления размером неявно созданных строк сетки.

```html
<div class="grid grid-flow-row **auto-rows-max**">
  <div>01</div>
  <div>02</div>
  <div>03</div>
</div>
```

## Использование пользовательских значений

### Настройка вашей темы

По умолчанию Tailwind включает четыре универсальных `grid-auto-rows`. Вы можете настроить эти значения, отредактировав `theme.gridAutoRows` или `theme.extend.gridAutoRows` в вашем файле `tailwind.config.js`.

```diff-js {{ filename: 'tailwind.config.js' }}
  module.exports = {
    theme: {
      extend: {
+       gridAutoRows: {
+         '2fr': 'minmax(0, 2fr)',
+       }
      }
    }
  }
```

Дополнительные сведения о настройке темы по умолчанию смотрите в документации [настройка темы](/docs/theme#customizing-the-default-theme).

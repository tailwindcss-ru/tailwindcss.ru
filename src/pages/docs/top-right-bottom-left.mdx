## Основы использования

### Размещение позиционированного элемента

Используйте утилиты `top-*`, `right-*`, `bottom-*`, `left-*` и `inset-*`, чтобы установить горизонтальное или вертикальное положение [позиционированного элемента](/docs/position).

```html {{ example: true }}
<div class="grid grid-cols-3 grid-rows-3 place-items-center gap-4 font-mono text-white text-sm font-bold leading-6">
  <div class="relative w-18 h-18 sm:w-32 sm:h-32 rounded-lg bg-stripes-purple">
    <div class="p-4 w-14 h-14 absolute left-0 top-0 flex items-center justify-center bg-purple-500 shadow-lg rounded-lg">01</div>
  </div>
  <div class="relative w-18 h-18 sm:w-32 sm:h-32 rounded-lg bg-stripes-purple">
    <div class="p-4 absolute inset-x-0 top-0 flex items-center justify-center bg-purple-500 shadow-lg rounded-lg">02</div>
  </div>
  <div class="relative w-18 h-18 sm:w-32 sm:h-32 rounded-lg bg-stripes-purple">
    <div class="p-4 w-14 h-14 absolute right-0 top-0 flex items-center justify-center bg-purple-500 shadow-lg rounded-lg">03</div>
  </div>
  <div class="relative w-18 h-18 sm:w-32 sm:h-32 rounded-lg bg-stripes-purple">
    <div class="p-4 w-14 absolute inset-y-0 left-0 flex items-center justify-center bg-purple-500 shadow-lg rounded-lg">04</div>
  </div>
  <div class="relative w-18 h-18 sm:w-32 sm:h-32 rounded-lg bg-stripes-purple">
    <div class="p-4 absolute inset-0 flex items-center justify-center bg-purple-500 shadow-lg rounded-lg">05</div>
  </div>
  <div class="relative w-18 h-18 sm:w-32 sm:h-32 rounded-lg bg-stripes-purple">
    <div class="p-4 w-14 absolute inset-y-0 right-0 flex items-center justify-center bg-purple-500 shadow-lg rounded-lg">06</div>
  </div>
  <div class="relative w-18 h-18 sm:w-32 sm:h-32 rounded-lg bg-stripes-purple">
    <div class="p-4 w-14 h-14 absolute left-0 bottom-0 flex items-center justify-center bg-purple-500 shadow-lg rounded-lg">07</div>
  </div>
  <div class="relative w-18 h-18 sm:w-32 sm:h-32 rounded-lg bg-stripes-purple">
    <div class="p-4 absolute inset-x-0 bottom-0 flex items-center justify-center bg-purple-500 shadow-lg rounded-lg">08</div>
  </div>
  <div class="relative w-18 h-18 sm:w-32 sm:h-32 rounded-lg bg-stripes-purple">
    <div class="p-4 w-14 h-14 absolute right-0 bottom-0 flex items-center justify-center bg-purple-500 shadow-lg rounded-lg">09</div>
  </div>
</div>
```

```html
<!-- Закрепить в верхнем левом углу -->
<div class="relative h-32 w-32 ...">
  <div class="absolute **left-0 top-0** h-16 w-16 ...">01</div>
</div>

<!-- Верхний край пролета -->
<div class="relative h-32 w-32 ...">
  <div class="absolute **inset-x-0 top-0** h-16 ...">02</div>
</div>

<!-- Закрепить в правом верхнем углу -->
<div class="relative h-32 w-32 ...">
  <div class="absolute **top-0 right-0** h-16 w-16 ...">03</div>
</div>

<!-- Притянут к левому краю -->
<div class="relative h-32 w-32 ...">
  <div class="absolute **inset-y-0 left-0** w-16 ...">04</div>
</div>

<!-- Заполнить весь родитель -->
<div class="relative h-32 w-32 ...">
  <div class="absolute **inset-0** ...">05</div>
</div>

<!-- Праввый край пролета -->
<div class="relative h-32 w-32 ...">
  <div class="absolute **inset-y-0 right-0** w-16 ...">06</div>
</div>

<!-- Закрепить в левом нижнем углу -->
<div class="relative h-32 w-32 ...">
  <div class="absolute **bottom-0 left-0** h-16 w-16 ...">07</div>
</div>

<!-- Нижний край пролета -->
<div class="relative h-32 w-32 ...">
  <div class="absolute **inset-x-0 bottom-0** h-16 ...">08</div>
</div>

<!-- Закрепить в правом нижнем углу -->
<div class="relative h-32 w-32 ...">
  <div class="absolute **bottom-0 right-0** h-16 w-16 ...">09</div>
</div>
```

### Использование отрицательных значений

Чтобы использовать отрицательное значение top/right/bottom/left, поставьте перед именем класса дефис, чтобы преобразовать его в отрицательное значение.

```html {{ example: true }}
<div class="flex justify-center font-mono text-white text-sm font-bold leading-6">
  <div class="relative w-18 h-18 sm:w-32 sm:h-32 rounded-lg bg-stripes-purple">
    <div class="p-4 w-14 h-14 absolute -left-4 -top-4 flex items-center justify-center bg-purple-500 shadow-lg rounded-lg"></div>
  </div>
</div>
```

```html
<div class="relative h-32 w-32 ...">
  <div class="absolute h-14 w-14 **-left-4 -top-4** ..."></div>
</div>
```

### Использование логических свойств

Используйте утилиты `start-*` и `end-*`, чтобы установить `inset-inline-start` и `inset-inline-end` [логические свойства](https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_Logical_Properties/Basic_concepts), которые сопоставляются с левой или правой стороной в зависимости от направления текста.

```html {{ example: true }}
<div class="grid grid-cols-2 gap-x-4 place-items-center">
  <div class="flex flex-col items-start gap-y-4">
    <p class="text-sm font-medium">Left-to-right</p>
    <div class="relative w-18 h-18 sm:w-32 sm:h-32 rounded-lg bg-stripes-purple">
      <div class="p-4 w-14 h-14 absolute top-0 left-0 flex items-center justify-center bg-purple-500 shadow-lg rounded-lg"></div>
    </div>
  </div>
  <div class="flex flex-col items-end gap-y-4">
    <p class="text-sm font-medium">Right-to-left</p>
    <div class="relative w-18 h-18 sm:w-32 sm:h-32 rounded-lg bg-stripes-purple">
      <div class="p-4 w-14 h-14 absolute top-0 right-0 flex items-center justify-center bg-purple-500 shadow-lg rounded-lg"></div>
    </div>
  </div>
</div>
```

```html
<div dir="**ltr**">
  <div class="relative h-32 w-32 ...">
    <div class="absolute h-14 w-14 top-0 **start-0** ..."></div>
  </div>
<div>

<div dir="**rtl**">
  <div class="relative h-32 w-32 ...">
    <div class="absolute h-14 w-14 top-0 **start-0** ..."></div>
  </div>
<div>
```

Для большего контроля вы также можете использовать [модификаторы LTR и RTL](/docs/hover-focus-and-other-states#rtl-support) для условного применения определенных стилей в зависимости от текущего направления текста.

## Использование пользовательских значений

### Настройка вашей темы

По умолчанию Tailwind предоставляет утилиты top/right/bottom/left/inset для комбинации [шкалы интервалов по умолчанию](/docs/customizing-spacing#default-spacing-scale), `auto`, `full`, а также некоторые дополнительные значения дроби.

Вы можете настроить масштаб интервалов, отредактировав `theme.spacing` или `theme.extend.spacing` в вашем файле `tailwind.config.js`.

```diff-js {{ filename: 'tailwind.config.js' }}
  module.exports = {
    theme: {
      extend: {
+       spacing: {
+         '3px': '3px',
+       }
      }
    }
  }
```

Кроме того, вы можете настроить только top/right/bottom/left/inset scale, отредактировав `theme.inset` или `theme.extend.inset` в вашем файле `tailwind.config.js`.

```diff-js {{ filename: 'tailwind.config.js' }}
  module.exports = {
    theme: {
      extend: {
+       inset: {
+         '3px': '3px',
+       }
      }
    }
  }
```

Дополнительные сведения о настройке темы по умолчанию смотрите в документации [настройка темы](/docs/theme#customizing-the-default-theme).

## Основы использования

### Установка максимальной высоты

Установите максимальную высоту элемента с помощью утилит `max-h-full` или `max-h-screen`.

```html
<div class="h-24 ...">
  <div class="h-48 **max-h-full** ...">
    <!-- ... -->
  </div>
</div>
```

## Использование пользовательских значений

### Настройка вашей темы

По умолчанию шкала максимальной высоты Tailwind использует комбинацию [шкалы интервалов по умолчанию](/docs/customizing-spacing), а также некоторых дополнительных значений, связанных с высотой.

Вы можете настроить масштаб интервалов, отредактировав `theme.spacing` или `theme.extend.spacing` в вашем файле `tailwind.config.js`.

```diff-js {{ filename: 'tailwind.config.js' }}
  module.exports = {
    theme: {
      extend: {
+       spacing: {
+         '128': '32rem',
+       }
      }
    }
  }
```

Кроме того, вы можете настроить только масштаб максимальной высоты, отредактировав `theme.maxHeight` или `theme.extend.maxHeight` в вашем файле `tailwind.config.js`.

```diff-js {{ filename: 'tailwind.config.js' }}
  module.exports = {
    theme: {
      extend: {
+       maxHeight: {
+         '128': '32rem',
+       }
      }
    }
  }
```

Дополнительные сведения о настройке темы по умолчанию смотрите в документации [настройка темы](/docs/theme#customizing-the-default-theme).

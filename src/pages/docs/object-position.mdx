## Использование пользовательских значений

### Настройка вашей темы

По умолчанию Tailwind предоставляет девять утилит для позиционирования объектов. Вы можете настроить эти значения, отредактировав `theme.objectPosition` или `theme.extend.objectPosition` в вашем файле `tailwind.config.js`.

```diff-js {{ filename: 'tailwind.config.js' }}
  module.exports = {
    theme: {
      extend: {
+       objectPosition: {
+         'center-bottom': 'center bottom',
+       }
      },
    }
  }
```

Дополнительные сведения о настройке темы по умолчанию смотрите в документации [настройка темы](/docs/theme#customizing-the-default-theme).

### Произвольные значения

<ArbitraryValues property="object-position">

```html
<img class="**object-[center_bottom]**" src="...">
```

</ArbitraryValues>

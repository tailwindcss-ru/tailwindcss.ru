## Основы использования

### Создание элемента в оттенках серого

Используйте утилиты `grayscale` и `grayscale-0`, чтобы указать, должен ли элемент отображаться в оттенках серого или в полноцветном.

```html {{ example: { p: 'none' } }}
<div class="flex justify-center sm:block overflow-scroll sm:overflow-visible scroll-p-8">
<div class="flex items-center justify-around gap-6 sm:gap-4 font-mono font-bold shrink-0 p-8">
  <div class="flex flex-col items-center shrink-0">
    <p class="font-medium text-sm text-slate-500 font-mono text-center mb-3 dark:text-slate-400">grayscale-0</p>
    <div class="relative grayscale-0">
      <img class="w-24 h-24 object-cover rounded-lg shadow-xl" src="https://images.unsplash.com/photo-1554629947-334ff61d85dc?ixid=MnwxMjA3fDB8MHxwaG90by1wYWdlfHx8fGVufDB8fHx8&ixlib=rb-1.2.1&auto=format&fit=crop&w=320&h=320&q=80" />
      <div class="absolute inset-0 ring-1 ring-inset ring-black/10 rounded-lg"></div>
    </div>
  </div>
  <div class="flex flex-col items-center shrink-0">
    <p class="font-medium text-sm text-slate-500 font-mono text-center mb-3 dark:text-slate-400">grayscale</p>
    <div class="relative grayscale">
      <img class="w-24 h-24 object-cover rounded-lg shadow-xl" src="https://images.unsplash.com/photo-1554629947-334ff61d85dc?ixid=MnwxMjA3fDB8MHxwaG90by1wYWdlfHx8fGVufDB8fHx8&ixlib=rb-1.2.1&auto=format&fit=crop&w=320&h=320&q=80" />
      <div class="absolute inset-0 ring-1 ring-inset ring-black/10 rounded-lg"></div>
    </div>
  </div>
</div>
</div>
```

```html
<div class="**grayscale-0** ...">
  <!-- ... -->
</div>
<div class="**grayscale** ...">
  <!-- ... -->
</div>
```

## Использование пользовательских значений

### Настройка вашей темы

По умолчанию Tailwind включает в себя несколько утилит общего назначения `grayscale`. Вы можете настроить эти значения, отредактировав `theme.grayscale` или `theme.extend.grayscale` в вашем файле `tailwind.config.js`.

```diff-js {{ filename: 'tailwind.config.js' }}
  module.exports = {
    theme: {
      extend: {
+       grayscale: {
+         50: '50%',
+       }
      }
    }
  }
```

Дополнительные сведения о настройке темы по умолчанию смотрите в документации [настройка темы](/docs/theme#customizing-the-default-theme).

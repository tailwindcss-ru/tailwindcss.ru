## Основы использования

### Привязка горизонтальной прокрутки

Используйте утилиту `snap-x`, чтобы включить привязку горизонтальной прокрутки к элементу.

Чтобы привязка прокрутки работала, вам также необходимо установить [выравнивание привязки прокрутки](/docs/scroll-snap-align) для дочерних элементов внутри элемента.

## Основы использования


Используйте `shrink`, чтобы позволить гибкому элементу сжиматься при необходимости:

```html {{ example: { resizable: true } }}
<div class="flex gap-4 text-white text-sm font-bold font-mono leading-6 bg-stripes-indigo rounded-lg">
  <div class="p-4 w-14 h-14 flex-none rounded-lg flex items-center justify-center bg-indigo-300 dark:bg-indigo-800 dark:text-indigo-400">
    01
  </div>
  <div class="p-4 w-64 shrink rounded-lg flex items-center justify-center bg-indigo-500 shadow-lg">
    02
  </div>
  <div class="p-4 w-14 h-14 flex-none rounded-lg hidden sm:flex items-center justify-center bg-indigo-300 dark:bg-indigo-800 dark:text-indigo-400">
    03
  </div>
</div>
```

```html
<div class="flex ...">
  <div class="flex-none w-14 h-14 ...">
    01
  </div>
  <div class="**shrink** w-64 h-14 ...">
    02
  </div>
  <div class="flex-none w-14 h-14 ...">
    03
  </div>
</div>
```

### Не сжиматься

Используйте `shrink-0`, чтобы предотвратить сжатие гибкого элемента:

```html {{ example: { resizable: true } }}
<div class="flex gap-4 text-white text-sm font-bold font-mono leading-6 bg-stripes-blue rounded-lg">
  <div class="p-4 flex-1 rounded-lg flex items-center justify-center bg-blue-300 dark:bg-blue-800 dark:text-blue-500">
    01
  </div>
  <div class="p-4 w-16 sm:w-64 shrink-0 rounded-lg flex items-center justify-center bg-blue-500 shadow-lg">
    02
  </div>
  <div class="p-4 flex-1 rounded-lg hidden sm:flex items-center justify-center bg-blue-300 dark:bg-blue-800 dark:text-blue-500">
    03
  </div>
</div>
```

```html
<div class="flex ...">
  <div class="flex-1 h-16 ...">
    01
  </div>
  <div class="**shrink-0** h-16 w-32 ...">
    02
  </div>
  <div class="flex-1 h-16 ...">
    03
  </div>
</div>
```

## Использование пользовательских значений

### Настройка вашей темы

По умолчанию Tailwind предоставляет два `shrink`. Вы можете настроить эти значения, отредактировав `theme.flexShrink` или `theme.extend.flexShrink` в вашем файле `tailwind.config.js`.

```diff-js {{ filename: 'tailwind.config.js' }}
  module.exports = {
    theme: {
      extend: {
+       flexShrink: {
+         2: '2'
+       }
      }
    }
  }
```

Дополнительные сведения о настройке темы по умолчанию смотрите в документации [настройка темы](/docs/theme#customizing-the-default-theme).

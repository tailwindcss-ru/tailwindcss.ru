## Основы использования

### Настройка смещения контура

Используйте утилиту `outline-*`, чтобы изменить смещение контура элемента.

```html {{ example: true }}
<div class="grid grid-cols-1 gap-8 sm:grid-cols-3 sm:gap-4 font-bold text-white text-sm text-center">
  <div class="flex flex-col items-center shrink-0">
    <p class="font-medium text-sm text-slate-500 font-mono text-center mb-3 dark:text-slate-400">outline-offset-0</p>
    <button class="px-4 py-2 font-semibold text-sm bg-white text-slate-700 border border-slate-300 rounded-md shadow-sm outline outline-2 outline-offset-0 outline-sky-500 dark:bg-slate-700 dark:text-slate-200 dark:border-transparent">Кнопка А</button>
  </div>
  <div class="flex flex-col items-center shrink-0">
    <p class="font-medium text-sm text-slate-500 font-mono text-center mb-3 dark:text-slate-400">outline-offset-2</p>
    <button class="px-4 py-2 font-semibold text-sm bg-white text-slate-700 border border-slate-300 rounded-md shadow-sm outline outline-2 outline-offset-2 outline-sky-500 dark:bg-slate-700 dark:text-slate-200 dark:border-transparent">Кнопка Б</button>
  </div>
  <div class="flex flex-col items-center shrink-0">
    <p class="font-medium text-sm text-slate-500 font-mono text-center mb-3 dark:text-slate-400">outline-offset-4</p>
    <button class="px-4 py-2 font-semibold text-sm bg-white text-slate-700 border border-slate-300 rounded-md shadow-sm outline outline-2 outline-offset-4 outline-sky-500 dark:bg-slate-700 dark:text-slate-200 dark:border-transparent">Кнопка В</button>
  </div>
</div>
```

```html
<button class="outline **outline-offset-0** ...">Кнопка А</button>
<button class="outline **outline-offset-2** ...">Кнопка Б</button>
<button class="outline **outline-offset-4** ...">Кнопка В</button>
```

## Использование пользовательских значений

### Настройка вашей темы

Вы можете настроить утилиты `outline-offset-*`, отредактировав `theme.outlineOffset` или `theme.extend.outlineOffset` в файле `tailwind.config.js`.

```diff-js {{ filename: 'tailwind.config.js' }}
  module.exports = {
    theme: {
      extend: {
+       outlineOffset: {
+         3: '3px',
+       }
      }
    }
  }
```

Дополнительные сведения о настройке темы по умолчанию смотрите в документации [настройка темы](/docs/theme#customizing-the-default-theme).

## Основы использования

### Настройка цвета смещения кольца

Используйте утилиты `ring-offset-{color}`, чтобы изменить цвет смещения кольца. Обычно это делается, чтобы попытаться сопоставить смещение с родительским цветом фона, поскольку истинные смещения тени блока фактически невозможны в CSS.

<Example>
  <div class="grid place-content-center text-white text-sm text-center font-bold leading-6">
    <div class="flex flex-col items-center shrink-0">
      <p class="font-medium text-sm font-mono text-slate-500 text-center mb-3 dark:text-slate-400">ring-offset-purple-100</p>
      <button class="px-4 py-2 font-semibold text-sm bg-white text-slate-700 border border-slate-300 rounded-md shadow-sm ring-2 ring-offset-4 ring-offset-purple-100 ring-purple-500 dark:bg-slate-700 dark:text-slate-200 dark:border-transparent">Сохранить изменения</button>
    </div>
  </div>
</Example>

```html
<button class="ring-2 ring-purple-500 ring-offset-4 **ring-offset-purple-100** ...">
  Сохранить изменения
</button>
```


### Изменение непрозрачности

Управляйте непрозрачностью цвета смещения кольца элемента с помощью модификатора непрозрачности цвета.

```html
<button class="ring-2 ring-purple-500 ring-offset-4 ring-offset-purple-100**/50**"></button>
```

Вы можете использовать любое значение, определенное в вашей [шкале непрозрачности](/docs/opacity), или использовать произвольные значения, если вам нужно отклониться от маркеров вашего дизайна.

```html
<button class="ring-2 ring-purple-500 ring-offset-4 ring-offset-purple-100**/[.55]**"></button>
```

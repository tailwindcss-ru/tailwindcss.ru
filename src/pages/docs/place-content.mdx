## Основы использования

### Center

Используйте `place-content-center`, чтобы упаковать элементы в центре оси блока:

```html {{ example: true }}
<div class="font-mono text-white text-sm font-bold leading-6 bg-stripes-sky rounded-lg">
  <div class="grid grid-cols-[repeat(2,56px)] place-content-center h-56 gap-4">
    <div class="p-4 w-14 h-14 flex items-center justify-center shadow-lg rounded-lg bg-sky-500">01</div>
    <div class="p-4 w-14 h-14 flex items-center justify-center shadow-lg rounded-lg bg-sky-500">02</div>
    <div class="p-4 w-14 h-14 flex items-center justify-center shadow-lg rounded-lg bg-sky-500">03</div>
    <div class="p-4 w-14 h-14 flex items-center justify-center shadow-lg rounded-lg bg-sky-500">04</div>
  </div>
</div>
```

```html
<div class="grid grid-cols-2 gap-4 **place-content-center** h-48 ...">
  <div>01</div>
  <div>02</div>
  <div>03</div>
  <div>04</div>
</div>
```

### Start

Используйте `place-content-start`, чтобы упаковать элементы напротив начала оси блока:

```html {{ example: true }}
<div class="font-mono text-white text-sm font-bold leading-6 bg-stripes-fuchsia rounded-lg">
  <div class="grid grid-cols-[repeat(2,56px)] place-content-start h-56 gap-4">
    <div class="p-4 w-14 h-14 flex items-center justify-center shadow-lg rounded-lg bg-fuchsia-500">01</div>
    <div class="p-4 w-14 h-14 flex items-center justify-center shadow-lg rounded-lg bg-fuchsia-500">02</div>
    <div class="p-4 w-14 h-14 flex items-center justify-center shadow-lg rounded-lg bg-fuchsia-500">03</div>
    <div class="p-4 w-14 h-14 flex items-center justify-center shadow-lg rounded-lg bg-fuchsia-500">04</div>
  </div>
</div>
```

```html
<div class="grid grid-cols-2 gap-4 **place-content-start** h-48 ...">
  <div>01</div>
  <div>02</div>
  <div>03</div>
  <div>04</div>
</div>
```

### End

Используйте `place-content-end`, чтобы упаковать элементы до конца оси блока:

```html {{ example: true }}
<div class="font-mono text-white text-sm font-bold leading-6 bg-stripes-indigo rounded-lg">
  <div class="grid grid-cols-[repeat(2,56px)] place-content-end h-56 gap-4">
    <div class="p-4 w-14 h-14 flex items-center justify-center shadow-lg rounded-lg bg-indigo-500">01</div>
    <div class="p-4 w-14 h-14 flex items-center justify-center shadow-lg rounded-lg bg-indigo-500">02</div>
    <div class="p-4 w-14 h-14 flex items-center justify-center shadow-lg rounded-lg bg-indigo-500">03</div>
    <div class="p-4 w-14 h-14 flex items-center justify-center shadow-lg rounded-lg bg-indigo-500">04</div>
  </div>
</div>
```

```html
<div class="grid grid-cols-2 gap-4 **place-content-end** h-48 ...">
  <div>01</div>
  <div>02</div>
  <div>03</div>
  <div>04</div>
</div>
```

### Space between

Используйте `place-content-between` для распределения элементов сетки по оси блока так, чтобы между каждой строкой на оси блока оставалось равное пространство.

```html {{ example: true }}
<div class="font-mono text-white text-sm font-bold leading-6 bg-stripes-purple rounded-lg">
  <div class="grid grid-cols-[repeat(2,56px)] place-content-between h-56 gap-4">
    <div class="p-4 w-14 h-14 flex items-center justify-center shadow-lg rounded-lg bg-purple-500">01</div>
    <div class="p-4 w-14 h-14 flex items-center justify-center shadow-lg rounded-lg bg-purple-500">02</div>
    <div class="p-4 w-14 h-14 flex items-center justify-center shadow-lg rounded-lg bg-purple-500">03</div>
    <div class="p-4 w-14 h-14 flex items-center justify-center shadow-lg rounded-lg bg-purple-500">04</div>
  </div>
</div>
```

```html
<div class="grid grid-cols-2 gap-4 **place-content-between** h-48 ...">
  <div>01</div>
  <div>02</div>
  <div>03</div>
  <div>04</div>
</div>
```

### Space around

Используйте `place-content-around`, чтобы распределить элементы сетки так, чтобы вокруг каждой строки на оси блока было одинаковое пространство:

```html {{ example: true }}
<div class="font-mono text-white text-sm font-bold leading-6 bg-stripes-cyan rounded-lg">
  <div class="grid grid-cols-[repeat(2,56px)] place-content-around h-56 gap-x-4">
    <div class="p-4 w-14 h-14 flex items-center justify-center shadow-lg rounded-lg bg-cyan-500">01</div>
    <div class="p-4 w-14 h-14 flex items-center justify-center shadow-lg rounded-lg bg-cyan-500">02</div>
    <div class="p-4 w-14 h-14 flex items-center justify-center shadow-lg rounded-lg bg-cyan-500">03</div>
    <div class="p-4 w-14 h-14 flex items-center justify-center shadow-lg rounded-lg bg-cyan-500">04</div>
  </div>
</div>
```

```html
<div class="grid grid-cols-2 gap-4 **place-content-around** h-48 ...">
  <div>01</div>
  <div>02</div>
  <div>03</div>
  <div>04</div>
</div>
```

### Space evenly

Используйте `place-content-evenly` для распределения элементов сетки так, чтобы они были равномерно распределены по оси блока:

```html {{ example: true }}
<div class="font-mono text-white text-sm font-bold leading-6 bg-stripes-violet rounded-lg">
  <div class="grid grid-cols-[repeat(2,56px)] place-content-evenly h-56">
    <div class="p-4 w-14 h-14 flex items-center justify-center shadow-lg rounded-lg bg-violet-500">01</div>
    <div class="p-4 w-14 h-14 flex items-center justify-center shadow-lg rounded-lg bg-violet-500">02</div>
    <div class="p-4 w-14 h-14 flex items-center justify-center shadow-lg rounded-lg bg-violet-500">03</div>
    <div class="p-4 w-14 h-14 flex items-center justify-center shadow-lg rounded-lg bg-violet-500">04</div>
  </div>
</div>
```

```html
<div class="grid grid-cols-2 gap-4 **place-content-evenly** h-48 ...">
  <div>01</div>
  <div>02</div>
  <div>03</div>
  <div>04</div>
</div>
```

### Stretch

Используйте `place-content-stretch`, чтобы растянуть элементы сетки по их областям сетки на оси блока:

```html {{ example: true }}
<div class="font-mono text-white text-sm font-bold leading-6 bg-stripes-pink rounded-lg">
  <div class="grid grid-cols-2 place-content-stretch h-56 gap-4">
    <div class="p-4 flex items-center justify-center shadow-lg rounded-lg bg-pink-500">01</div>
    <div class="p-4 flex items-center justify-center shadow-lg rounded-lg bg-pink-500">02</div>
    <div class="p-4 flex items-center justify-center shadow-lg rounded-lg bg-pink-500">03</div>
    <div class="p-4 flex items-center justify-center shadow-lg rounded-lg bg-pink-500">04</div>
  </div>
</div>
```

```html
<div class="grid grid-cols-2 gap-4 **place-content-stretch** h-48 ...">
  <div>01</div>
  <div>02</div>
  <div>03</div>
  <div>04</div>
</div>
```

## Основы использования

### Изменение прозрачности элемента

Используйте утилиту `opacity-*` для управления непрозрачностью элемента.

```html {{ example: true }}
<div class="flex flex-col sm:flex-row items-center justify-center gap-8 sm:gap-16 text-white text-sm font-bold leading-6">
  <div class="flex flex-col items-center shrink-0">
    <p class="font-medium text-sm text-slate-500 font-mono text-center mb-3 dark:text-slate-400">opacity-100</p>
    <button class="px-4 py-2 font-semibold text-sm bg-indigo-500 text-white rounded-md shadow-sm opacity-100">Кнопка А</button>
  </div>
  <div class="flex flex-col items-center shrink-0">
    <p class="font-medium text-sm text-slate-500 font-mono text-center mb-3 dark:text-slate-400">opacity-75</p>
    <button class="px-4 py-2 font-semibold text-sm bg-indigo-500 text-white rounded-md shadow-sm opacity-75">Кнопка Б</button>
  </div>
  <div class="flex flex-col items-center shrink-0">
    <p class="font-medium text-sm text-slate-500 font-mono text-center mb-3 dark:text-slate-400">opacity-50</p>
    <button class="px-4 py-2 font-semibold text-sm bg-indigo-500 text-white rounded-md shadow-sm opacity-50">Кнопка В</button>
  </div>
  <div class="flex flex-col items-center shrink-0">
    <p class="font-medium text-sm text-slate-500 font-mono text-center mb-3 dark:text-slate-400">opacity-25</p>
    <button class="px-4 py-2 font-semibold text-sm bg-indigo-500 text-white rounded-md shadow-sm opacity-25">Кнопка Г</button>
  </div>
</div>
```

```html
<button class="bg-indigo-500 **opacity-100** ..."></button>
<button class="bg-indigo-500 **opacity-75** ..."></button>
<button class="bg-indigo-500 **opacity-50** ..."></button>
<button class="bg-indigo-500 **opacity-25** ..."></button>
```

## Использование пользовательских значений

### Настройка вашей темы

По умолчанию Tailwind предоставляет несколько утилит непрозрачности для общего использования. Вы можете настроить эти значения, отредактировав `theme.opacity` или `theme.extend.opacity` в вашем файле `tailwind.config.js`.

```diff-js {{ filename: 'tailwind.config.js' }}
  module.exports = {
    theme: {
      extend: {
+       opacity: {
+         '67': '.67',
+       }
      }
    }
  }
```

Дополнительные сведения о настройке темы по умолчанию смотрите в документации [настройка темы](/docs/theme#customizing-the-default-theme).

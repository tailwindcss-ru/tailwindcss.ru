## Основы использования

### Установка цвета заливки

Используйте утилиты `fill-*` для изменения цвета заливки SVG.

```html {{ example: true }}
<div class="flex items-center justify-center">
  <svg class="fill-blue-500 h-12" viewBox="0 0 46 48" xmlns="http://www.w3.org/2000/svg"><path fill-rule="evenodd" clip-rule="evenodd" d="M23.0002 0C12.5068 0 4.00017 8.50659 4.00017 19V32.5335C4.00017 32.8383 3.9145 33.1371 3.75292 33.3956L0.912672 37.94C0.0801118 39.2721 1.0378 41 2.60867 41H43.3917C44.9625 41 45.9202 39.2721 45.0877 37.94L42.2474 33.3956C42.0858 33.1371 42.0002 32.8383 42.0002 32.5335V19C42.0002 8.50659 33.4936 0 23.0002 0ZM23.0002 48C20.2388 48 18.0002 45.7614 18.0002 43H28.0002C28.0002 45.7614 25.7616 48 23.0002 48Z"/></svg>
</div>
```

```html
<svg class="**fill-blue-500** ...">
  <!-- ... -->
</svg>
```

Это может быть полезно для стилизации наборов иконок, таких как [Heroicons](https://heroicons.com).

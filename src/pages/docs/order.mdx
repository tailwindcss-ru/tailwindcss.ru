## Основы использования

### Порядок флекс и грид элементов

Используйте утилиты `order-*` для рендеринга гибких элементов и элементов сетки в другом порядке, чем они отображаются в DOM.

```html {{ example: true }}
<div class="flex justify-between font-mono text-white text-sm font-bold leading-6">
  <div class="w-14 h-14 rounded-lg flex items-center justify-center bg-sky-500 shadow-lg order-last">01</div>
  <div class="w-14 h-14 rounded-lg flex items-center justify-center bg-sky-500 shadow-lg">02</div>
  <div class="w-14 h-14 rounded-lg flex items-center justify-center bg-sky-500 shadow-lg">03</div>
</div>
```

```html
<div class="flex justify-between ...">
  <div class="**order-last**">01</div>
  <div>02</div>
  <div>03</div>
</div>
```

### Использование отрицательных значений

Чтобы использовать значение отрицательного порядка, поставьте перед именем класса дефис, чтобы преобразовать его в отрицательное значение.

```html
<div class="**-order-1**">
  <!-- ... -->
</div>
```

## Использование пользовательских значений

### Настройка вашей темы

По умолчанию Tailwind предоставляет утилиты для `order-first`, `order-last`, `order-none` и утилит с номерами от 1 до 12. Вы можете настроить эти значения, отредактировав `theme.order` или `theme.extend.order` в вашем файле `tailwind.config.js`.

```diff-js {{ filename: 'tailwind.config.js' }}
  module.exports = {
    theme: {
      extend: {
+       order: {
+         '13': '13'
+       }
      }
    }
  }
```

Дополнительные сведения о настройке темы по умолчанию смотрите в документации [настройка темы](/docs/theme#customizing-the-default-theme).

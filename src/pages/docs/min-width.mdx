## Основы использования

### Установка минимальной ширины

Установите минимальную ширину элемента с помощью утилит `min-w-{width}`.

```html
<span class="**min-w-full** ...">
  <!-- ... -->
</span>
```

## Использование пользовательских значений

### Настройка вашей темы

Вы можете настроить масштаб `min-width`, отредактировав `theme.minWidth` или `theme.extend.minWidth` в вашем файле `tailwind.config.js`.

```diff-js tailwind.config.js
  module.exports = {
    theme: {
+     minWidth: {
+       '1/2': '50%',
+     }
    }
  }
```

Дополнительные сведения о настройке темы по умолчанию смотрите в документации [настройка темы](/docs/theme#customizing-the-default-theme).

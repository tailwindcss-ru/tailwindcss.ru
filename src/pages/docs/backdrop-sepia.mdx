## Основы использования

### Добавление фильтров сепии для фона

Используйте утилиты `backdrop-sepia` и `backdrop-sepia-0` для управления тем, должен ли фон элемента отображаться как сепия или в полном цвете.

    <p class="font-medium text-sm text-slate-500 font-mono text-center mb-3 dark:text-slate-400">backdrop-sepia-0</p>
    <div class="relative">

## Использование пользовательских значений

### Настройка вашей темы

По умолчанию Tailwind включает в себя несколько утилит общего назначения `backdrop-sepia`. Вы можете настроить эти значения, отредактировав `theme.backdropSepia` или `theme.extend.backdropSepia` в вашем файле `tailwind.config.js`.

```diff-js {{ filename: 'tailwind.config.js' }}
  module.exports = {
    theme: {
      extend: {
+       backdropSepia: {
+         25: '.25',
+         75: '.75',
+       }
      }
    }
  }
```

Дополнительные сведения о настройке темы по умолчанию смотрите в документации [настройка темы](/docs/theme#customizing-the-default-theme).

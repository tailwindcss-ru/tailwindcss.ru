### Добавление на основе ширины столбца

Используйте утилиты `columns-{width}` для установки идеальной ширины столбца для содержимого внутри элемента, при этом количество столбцов (счетчик) автоматически регулируется в соответствии с этим значением.

Этот масштаб "t-shirt" такой же, как и масштаб [max-width](/docs/max-width), с добавлением `2xs` и `3xs`, поскольку могут быть желательны меньшие столбцы.


## Использование пользовательских значений

### Настройка вашей темы

По умолчанию Tailwind предоставляет шкалу подсчета столбцов `1-12`, а также шкалу t-shirt от `3xs-7xl`. Вы можете настроить эти значения, отредактировав `theme.columns` или `theme.extend.columns` в вашем файле `tailwind.config.js`.

```diff-js {{ filename: 'tailwind.config.js' }}
  module.exports = {
    theme: {
      extend: {
+       columns: {
+         '4xs': '14rem',
+       }
      },
    }
  }
```

Дополнительные сведения о настройке темы по умолчанию смотрите в документации [настройка темы](/docs/theme#customizing-the-default-theme).

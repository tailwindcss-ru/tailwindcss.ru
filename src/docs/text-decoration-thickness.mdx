## Примеры

### Базовый пример

Используйте утилиты `decoration-<number>` типа `decoration-2` и `decoration-4` для изменения толщины [текстового украшения](/docs/text-decoration-line) элемента:

        <p className="text-lg font-medium underline decoration-1">Быстрая коричневая лиса перепрыгивает через ленивую собаку.</p>
      </div>
      <div>
        <div className="mb-3 font-mono text-xs font-medium text-gray-500 dark:text-gray-400">decoration-2</div>

      </div>
      <div>
        <div className="mb-3 font-mono text-xs font-medium text-gray-500 dark:text-gray-400">decoration-4</div>

      </div>
    </div>
  }
</Example>

```html
<!-- [!code classes:decoration-1] -->
<p class="underline decoration-1">Быстрая коричневая лиса...</p>
<!-- [!code classes:decoration-2] -->
<p class="underline decoration-2">Быстрая коричневая лиса...</p>
<!-- [!code classes:decoration-4] -->
<p class="underline decoration-4">Быстрая коричневая лиса...</p>
```

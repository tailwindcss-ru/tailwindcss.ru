## Примеры

### Сглаживание в оттенках серого

Используйте утилиту `antialiased` для отображения текста с использованием сглаживания в оттенках серого:

<Figure>

<Example>
  {
    <p className="text-center text-lg font-medium text-gray-900 antialiased dark:text-gray-200">
      Быстрая коричневая лиса перепрыгивает через ленивую собаку.
    </p>
  }
</Example>

```html
<!-- [!code classes:antialiased] -->
<p class="antialiased ...">Быстрая коричневая лиса перепрыгивает через ленивую собаку ...</p>
```

</Figure>

### Субпиксельное сглаживание

Используйте утилиту `subpixel-antialiased` для отображения текста с использованием субпиксельного сглаживания:

<Figure>

<Example>
  {
    <p className="text-center text-lg font-medium text-gray-900 subpixel-antialiased dark:text-gray-200">
      Быстрая коричневая лиса перепрыгивает через ленивую собаку.
    </p>
  }
</Example>

```html
<!-- [!code classes:subpixel-antialiased] -->
<p class="subpixel-antialiased ...">Быстрая коричневая лиса перепрыгивает через ленивую собаку ...</p>
```

</Figure>

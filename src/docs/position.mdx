## Примеры

### Статическое позиционирование элементов

Используйте утилиту `static` для позиционирования элемента в соответствии с нормальным потоком документа:

<Figure>

<Example>
  {
    <div className="relative text-sm leading-6 font-medium">
      <div className="rounded-lg border border-purple-700/10 bg-purple-400/20 p-4 dark:border-0 dark:bg-sky-900/70">
        <div className="static h-32 border border-purple-700/10 bg-purple-400/20 p-4 dark:border-0 dark:bg-sky-400/20">
          <p className="text-purple-700 dark:text-sky-200">Статический родитель</p>
          <div className="absolute bottom-0 left-0 rounded-lg bg-purple-500 p-4 text-white shadow-lg dark:bg-sky-500">
            <p>Абсолютный дочерний элемент</p>
          </div>
        </div>
      </div>
    </div>
  }
</Example>

```html
<!-- [!code classes:static] -->
<div class="static ...">
  <p>Статический родитель</p>
  <div class="absolute bottom-0 left-0 ...">
    <p>Абсолютный дочерний элемент</p>
  </div>
</div>
```

</Figure>

При статически позиционированных элементах любые [смещения](/docs/top-right-bottom-left) будут игнорироваться, и элемент не будет действовать как позиционная ссылка для абсолютно позиционированных дочерних элементов.

### Относительное позиционирование элементов

Используйте утилиту `relative` для позиционирования элемента в соответствии с нормальным потоком документа:

<Figure>

<Example>
  {
    <div className="relative text-sm leading-6 font-medium">
      <div className="rounded-lg border border-sky-700/10 bg-sky-400/20 p-4 dark:border-0 dark:bg-blue-900/70">
        <div className="relative h-32 border border-sky-700/10 bg-sky-400/20 p-4 dark:border-0 dark:bg-blue-400/20">
          <p className="text-sky-700 dark:text-white">Относительный родитель</p>
          <div className="absolute bottom-0 left-0 rounded-lg bg-sky-500 p-4 text-white shadow-lg dark:bg-blue-500">
            <p>Абсолютный дочерний элемент</p>
          </div>
        </div>
      </div>
    </div>
  }
</Example>

```html
<!-- [!code classes:relative] -->
<div class="relative ...">
  <p>Относительный родитель</p>
  <div class="absolute bottom-0 left-0 ...">
    <p>Абсолютный дочерний элемент</p>
  </div>
</div>
```

</Figure>

При относительно позиционированных элементах любые [смещения](/docs/top-right-bottom-left) рассчитываются относительно нормальной позиции элемента, и элемент будет действовать как позиционная ссылка для абсолютно позиционированных дочерних элементов.

### Абсолютное позиционирование элементов

Используйте утилиту `absolute` для позиционирования элемента _вне_ нормального потока документа, заставляя соседние элементы вести себя так, как будто элемент не существует:

<Figure>

<Example>
  {
    <div className="space-y-8">
      <div>
        <p className="mb-4 text-sm font-medium text-gray-500">Со статическим позиционированием</p>

            <div className="static flex h-32 flex-col justify-between border border-indigo-700/10 bg-indigo-400/20 p-4 dark:border-0">
              <p className="text-indigo-700 dark:text-indigo-200">Статический родитель</p>
              <div className="flex gap-4">
                <div className="bottom-0 left-0 rounded-lg bg-indigo-500 p-4 text-white shadow-lg">
                  <p>Статический дочерний элемент?</p>
                </div>
                <div className="rounded-lg bg-indigo-100 p-4 text-indigo-600 shadow-lg">

                </div>

            </div>
          </div>
        </div>
      </div>

            <div className="static flex h-32 flex-col justify-between border border-indigo-700/10 bg-indigo-400/20 p-4 dark:border-0">
              <p className="text-indigo-700 dark:text-indigo-200">Статический родитель</p>
              <div className="flex gap-4">
                <div className="absolute top-0 right-0 rounded-lg bg-indigo-500 p-4 text-white shadow-lg">
                  <p>Абсолютный дочерний элемент</p>
                </div>
                <div className="rounded-lg bg-indigo-100 p-4 text-indigo-600 shadow-lg">
                  <p>Статический соседний элемент</p>
                </div>
              </div>
            </div>
          </div>
        </div>
      </div>
    </div>
  }
</Example>

```html
<!-- [!code classes:absolute] -->
<div class="static ...">
  <!-- Статический родитель -->
  <div class="static ..."><p>Статический дочерний элемент</p></div>
  <div class="inline-block ..."><p>Статический соседний элемент</p></div>
  <!-- Статический родитель -->
  <div class="absolute ..."><p>Абсолютный дочерний элемент</p></div>
  <div class="inline-block ..."><p>Статический соседний элемент</p></div>
</div>
```

</Figure>

При абсолютно позиционированных элементах любые [смещения](/docs/top-right-bottom-left) рассчитываются относительно ближайшего родителя, который имеет позицию, отличную от `static`, и элемент будет действовать как позиционная ссылка для других абсолютно позиционированных дочерних элементов.

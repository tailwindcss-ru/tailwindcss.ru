## Примеры

### Разрешение переноса текста

Используйте утилиту `text-wrap` для переноса переполняющегося текста на несколько строк в логических точках текста:

<Figure>

<Example padding={false}>
  {
    <div className="mx-auto grid max-w-xs gap-4 border-x border-x-pink-400/30 py-8 text-gray-700 dark:text-gray-400">
      <div className="text-xl font-semibold text-gray-900 dark:text-white">Любимый суповой киоск Манхэттена закрывается</div>
      <p className="text-sm/6">
        Жители Нью-Йорка встречают зимний холод с меньшим теплом в этом году, поскольку самый почитаемый суповой киоск города 
        неожиданно закрывается, следуя серии событий, которые оставили сообщество в недоумении.
      </p>
    </div>
  }
</Example>

```html
<!-- [!code classes:text-wrap] -->
<article class="text-wrap">
  <h3>Любимый суповой киоск Манхэттена закрывается</h3>
  <p>Жители Нью-Йорка встречают зимний холод...</p>
</article>
```

</Figure>

          Любимый суповой киоск Манхэттена закрывается
        </div>
        <p className="text-sm/6 text-nowrap">

          неожиданно закрывается, следуя серии событий, которые оставили сообщество в недоумении.
        </p>
      </div>
    </div>
  }
</Example>

```html
<!-- [!code classes:text-nowrap] -->
<article class="text-nowrap">
  <h3>Любимый суповой киоск Манхэттена закрывается</h3>
  <p>Жители Нью-Йорка встречают зимний холод...</p>
</article>
```

</Figure>

### Сбалансированный перенос текста

Используйте утилиту `text-balance` для равномерного распределения текста по каждой строке:

<Figure>

<Example padding={false}>
  {
    <div className="mx-auto grid max-w-xs gap-4 border-x border-x-pink-400/30 py-8 text-gray-700 dark:text-gray-400">
      <div className="text-xl font-semibold text-balance text-gray-900 dark:text-white">
        Любимый суповой киоск Манхэттена закрывается
      </div>
      <p className="text-sm/6">
        Жители Нью-Йорка встречают зимний холод с меньшим теплом в этом году, поскольку самый почитаемый суповой киоск города 
        неожиданно закрывается, следуя серии событий, которые оставили сообщество в недоумении.
      </p>
    </div>
  }
</Example>

```html
<!-- [!code classes:text-balance] -->
<article>
  <h3 class="text-balance">Любимый суповой киоск Манхэттена закрывается</h3>
  <p>Жители Нью-Йорка встречают зимний холод...</p>
</article>
```

</Figure>

По соображениям производительности браузеры ограничивают балансировку текста блоками, которые составляют ~6 строк или меньше, что делает его наиболее подходящим для заголовков.

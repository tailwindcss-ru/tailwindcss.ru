## Примеры

### Подчеркивание текста

Используйте утилиту `underline` для добавления подчеркивания к тексту элемента:

<Figure>

<Example>
  {
    <p className="text-center text-lg font-medium text-gray-900 underline dark:text-gray-200">
      Быстрая коричневая лиса перепрыгивает через ленивую собаку.
    </p>
  }
</Example>

```html
<!-- [!code classes:underline] -->
<p class="underline">Быстрая коричневая лиса...</p>
```

</Figure>

### Добавление надчеркивания к тексту

Используйте утилиту `overline` для добавления надчеркивания к тексту элемента:

<Figure>

<Example>
  {
    <p className="text-center text-lg font-medium text-gray-900 overline dark:text-gray-200">
      Быстрая коричневая лиса перепрыгивает через ленивую собаку.
    </p>
  }
</Example>

```html
<!-- [!code classes:overline] -->
<p class="overline">Быстрая коричневая лиса...</p>
```

</Figure>

### Добавление зачеркивания к тексту

Используйте утилиту `line-through` для добавления зачеркивания к тексту элемента:

<Figure>

<Example>
  {
    <p className="text-center text-lg font-medium text-gray-900 line-through dark:text-gray-200">

  }
</Example>

```html

```

</Figure>

### Удаление линии из текста

Используйте утилиту `no-underline` для удаления линии из текста элемента:

</Example>


</Figure>

### Применение при наведении

<TargetingSpecificStates property="text-decoration-line">

<Figure hint="Наведите на текст, чтобы увидеть ожидаемое поведение">

<Example>
  {
    <div className="text-center text-lg font-medium text-gray-900 no-underline dark:text-gray-200">
      Быстрая{" "}
      <a
        href="https://en.wikipedia.org/wiki/The_quick_brown_fox_jumps_over_the_lazy_dog"
        target="blank"
        className="text-sky-600 no-underline hover:underline dark:text-sky-400"
      >
        коричневая лиса
      </a>{" "}
      перепрыгивает через ленивую собаку.
    </div>
  }
</Example>

```html
<!-- [!code classes:hover:underline] -->
<p>Быстрая <a href="..." class="no-underline hover:underline ...">коричневая лиса</a> перепрыгивает через ленивую собаку.</p>
```

</Figure>

</TargetingSpecificStates>

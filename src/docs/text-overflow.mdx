## Примеры

### Обрезка текста

Используйте утилиту `truncate` для предотвращения переноса текста и обрезки переполняющегося текста с многоточием (…) при необходимости:

<Figure>

<Example padding={false}>
  {
    <p className="mx-auto max-w-xs truncate border-x border-x-pink-400/30 py-8 text-gray-900 dark:text-gray-200">
      Самое длинное слово в любом из основных словарей английского языка - это{" "}
      <span className="font-bold">пневмоультрамикроскопикосиковулканокониоз,</span> слово, которое относится к заболеванию легких, 
      полученному от вдыхания очень мелких частиц кремнезема, особенно из вулкана; с медицинской точки зрения это то же самое, что и силикоз.
    </p>
  }
</Example>

```html
<!-- [!code classes:truncate] -->
<p class="truncate">Самое длинное слово в любом из основных...</p>
```

</Figure>

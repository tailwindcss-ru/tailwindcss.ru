## Кастомизация вашей темы

<CustomizingYourTheme
  utility="font"
  name="font family"
  customName="display"
  customValue='"Oswald", "sans-serif"'
>

Вы также можете предоставить значения по умолчанию для `font-feature-settings` и `font-variation-settings` для семейства шрифтов:

```css
@theme {
  --font-display: "Oswald", "sans-serif";
  --font-display--font-feature-settings: "cv02", "cv03", "cv04", "cv11"; /* [!code highlight] */
  --font-display--font-variation-settings: "opsz" 32; /* [!code highlight] */
}
```

При необходимости используйте правило [@font-face](https://developer.mozilla.org/en-US/docs/Web/CSS/@font-face) для загрузки пользовательских шрифтов:

```css
@font-face {
  font-family: Oswald;
  font-style: normal;
  font-weight: 200 700;
  font-display: swap;
  src: url("/fonts/Oswald.woff2") format("woff2");
}
```

Если вы загружаете шрифт из сервиса типа [Google Fonts](https://fonts.google.com/), убедитесь, что `@import` находится в самом верху вашего CSS файла:

```css
@import url("https://fonts.googleapis.com/css2?family=Roboto&display=swap"); /* [!code highlight] */
@import "tailwindcss";

@theme {
  --font-roboto: "Roboto", sans-serif; /* [!code highlight] */
}
```

Браузеры требуют, чтобы операторы `@import` шли перед любыми другими правилами, поэтому URL импорты должны быть выше импортов типа `@import "tailwindcss"`, которые встраиваются в скомпилированный CSS.

</CustomizingYourTheme>

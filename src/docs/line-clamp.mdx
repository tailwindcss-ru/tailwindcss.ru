### Отмена ограничения строк

Используйте `line-clamp-none` для отмены ранее примененной утилиты ограничения строк:

```html
<!-- [!code classes:lg:line-clamp-none] -->
<p class="line-clamp-3 lg:line-clamp-none">
  <!-- ... -->
</p>
```

### Undoing line clamping

Use `line-clamp-none` to undo a previously applied line clamp utility:

```html
<!-- [!code classes:lg:line-clamp-none] -->
<p class="line-clamp-3 lg:line-clamp-none">
  <!-- ... -->
</p>
```

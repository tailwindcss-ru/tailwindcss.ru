## Примеры

### Установка акцентного цвета

Используйте утилиты, такие как `accent-rose-500` и `accent-lime-600`, чтобы изменить акцентный цвет элемента:

<Figure>

<Example>
  {
    <div className="flex flex-wrap justify-center gap-6">
      <label className="flex items-center space-x-2">
        <input type="checkbox" defaultChecked />
        <div className="text-sm font-semibold text-gray-900 dark:text-gray-200">Browser default</div>
      </label>
      <label className="flex items-center space-x-2">
        <input type="checkbox" className="accent-pink-500" defaultChecked />
        <div className="text-sm font-semibold text-gray-900 dark:text-gray-200">Customized</div>
      </label>
    </div>
  }
</Example>

```html
<!-- [!code classes:accent-pink-500] -->
<label>
  <input type="checkbox" checked />
  Браузер по умолчанию
</label>
<label>
  <input class="accent-pink-500" type="checkbox" checked />
  Кастомизированный
</label>
```

</Figure>

Это полезно для стилизации элементов, таких как чекбоксы и радиогруппы, позволяя переопределить стандартный цвет браузера.

### Изменение прозрачности

Используйте модификатор прозрачности цвета, чтобы управлять прозрачностью акцентного цвета элемента:

<Figure>

<Example>
  {
    <div className="flex flex-wrap justify-center gap-6">
      <label className="flex items-center space-x-2">
        <input type="checkbox" defaultChecked className="accent-purple-500/25" />
        <div className="text-sm font-semibold text-gray-900 dark:text-gray-200">accent-black/25</div>
      </label>
      <label className="flex items-center space-x-2">
        <input type="checkbox" className="accent-purple-500/75" defaultChecked />
        <div className="text-sm font-semibold text-gray-900 dark:text-gray-200">accent-black/75</div>
      </label>
    </div>
  }
</Example>

```html
<!-- [!code word:/25] -->
<!-- [!code word:/75] -->
<input class="accent-purple-500/25" type="checkbox" checked />
<input class="accent-purple-500/75" type="checkbox" checked />
```

</Figure>

Установка прозрачности акцентного цвета имеет ограниченную поддержку браузеров и на данный момент работает только в Firefox.

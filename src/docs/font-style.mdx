## Примеры

### Курсивный текст

Используйте утилиту `italic` для создания курсивного текста:

<Figure>

<Example>
  {
    <p className="text-center text-lg font-medium text-gray-900 italic dark:text-gray-200">
      Быстрая коричневая лиса перепрыгивает через ленивую собаку.
    </p>
  }
</Example>

```html
<!-- [!code classes:italic] -->
<p class="italic ...">Быстрая коричневая лиса перепрыгивает через ленивую собаку ...</p>
```

</Figure>

### Отображение текста обычным шрифтом

Используйте утилиту `not-italic` для отображения текста обычным шрифтом:

<Figure>

<Example>
  {
    <p className="text-center text-lg font-medium text-gray-900 not-italic dark:text-gray-200">
      Быстрая коричневая лиса перепрыгивает через ленивую собаку.
    </p>
  }
</Example>

```html
<!-- [!code classes:not-italic] -->
<p class="not-italic ...">Быстрая коричневая лиса перепрыгивает через ленивую собаку ...</p>
```

</Figure>

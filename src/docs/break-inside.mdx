## Примеры

### Базовый пример

Используйте утилиты вроде `break-inside-column` и `break-inside-avoid-page` для управления тем, как должен вести себя разрыв столбца или страницы внутри элемента:

```html
<!-- [!code classes:break-inside-avoid-column] -->
<div class="columns-2">
  <p>Well, let me tell you something, ...</p>
  <p class="break-inside-avoid-column">Sure, go ahead, laugh...</p>
  <p>Maybe we can live without...</p>
  <p>Look. If you think this is...</p>
</div>
```

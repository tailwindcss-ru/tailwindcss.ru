## Примеры

### Базовый пример

спользуйте утилиты вроде `break-before-column` и `break-before-page` для управления тем, как должен вести себя разрыв столбца или страницы перед элементом:

```html
<!-- [!code classes:break-before-column] -->
<div class="columns-2">
  <p>Well, let me tell you something, ...</p>
  <p class="break-before-column">Sure, go ahead, laugh...</p>
  <p>Maybe we can live without...</p>
  <p>Look. If you think this is...</p>
</div>
```

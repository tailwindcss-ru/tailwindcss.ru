## Примеры

### Изменение размера во всех направлениях

Используйте `resize` для того, чтобы сделать элемент изменяемым по горизонтали и вертикали:

<Figure hint="Перетащите ручку textarea в демо, чтобы увидеть ожидаемое поведение">

<Example>
  {
    <textarea
      rows="2"
      className="mx-auto block w-80 resize rounded-md p-2 text-sm text-gray-950 outline-1 outline-gray-900/10 focus:outline-2 focus:outline-gray-900 dark:bg-gray-950/25 dark:text-white dark:outline-1 dark:outline-white/5 dark:focus:outline-white/20"
    ></textarea>
  }
</Example>

```html
<!-- [!code classes:resize] -->
<textarea class="resize rounded-md ..."></textarea>
```

</Figure>

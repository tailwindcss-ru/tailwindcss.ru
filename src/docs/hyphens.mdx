## Примеры

### Предотвращение переноса

Используйте утилиту `hyphens-none` для предотвращения переноса слов, даже если используется предложение переноса строки `&shy;`:

<Figure>

<Example padding={false}>
  {
    <p className="mx-auto max-w-xs border-x border-x-pink-400/30 py-8 break-normal hyphens-none text-gray-900 dark:text-gray-200">
      Официально признанное словарем Дудена как самое длинное слово в немецком языке,{" "}
      <span className="font-bold" lang="de">
        Kraftfahrzeug&shy;haftpflichtversicherung
      </span>{" "}
      это 36-буквенное слово для страхования гражданской ответственности автотранспортных средств.
    </p>
  }
</Example>

```html
<!-- [!code classes:hyphens-none] -->
<!-- [!code word:&shy;] -->
<!-- prettier-ignore -->
<p class="hyphens-none">
  ... Kraftfahrzeug&shy;haftpflichtversicherung is a ...
</p>
```

</Figure>

### Ручной перенос

Используйте утилиту `hyphens-manual` для установки точек переноса только там, где используется предложение переноса строки `&shy;`:

<Figure>

<Example padding={false}>
  {
    <p className="mx-auto max-w-xs border-x border-x-pink-400/30 py-8 break-normal hyphens-manual text-gray-900 dark:text-gray-200">
      Официально признанное словарем Дудена как самое длинное слово в немецком языке,{" "}
      <span className="font-bold" lang="de">
        Kraftfahrzeug&shy;haftpflichtversicherung
      </span>{" "}
      это 36-буквенное слово для страхования гражданской ответственности автотранспортных средств.
    </p>
  }
</Example>

```html
<!-- [!code classes:hyphens-manual] -->
<!-- [!code word:&shy;] -->
<!-- prettier-ignore -->
<p class="hyphens-manual">
  ... Kraftfahrzeug&shy;haftpflichtversicherung is a ...
</p>
```

</Figure>

Это поведение браузера по умолчанию.

### Автоматический перенос

Используйте утилиту `hyphens-auto` для разрешения браузеру автоматически выбирать точки переноса на основе языка:

<Figure>

<Example padding={false}>
  {
    <p className="mx-auto max-w-xs border-x border-x-pink-400/30 py-8 break-normal hyphens-auto text-gray-900 dark:text-gray-200">
      Официально признанное словарем Дудена как самое длинное слово в немецком языке,{" "}
      <span className="font-bold" lang="de">
        Kraftfahrzeughaftpflichtversicherung
      </span>{" "}
      это 36-буквенное слово для страхования гражданской ответственности автотранспортных средств.
    </p>
  }
</Example>

```html
<!-- [!code classes:hyphens-auto] -->
<!-- [!code word:lang="de"] -->
<!-- prettier-ignore -->
<p class="hyphens-auto" lang="de">
  ... Kraftfahrzeughaftpflichtversicherung is a ...
</p>
```

</Figure>

Предложение переноса строки `&shy;` будет предпочтительнее автоматических точек переноса.

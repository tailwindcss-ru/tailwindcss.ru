## Примеры

### Выравнивание по базовой линии

Используйте утилиту `align-baseline` для выравнивания базовой линии элемента с базовой линией его родителя:

<Figure>

<Example>
  {
    <div className="relative mx-auto max-w-md leading-none">
      <span className="inline-block h-8 w-0 align-baseline">
        <span className="absolute top-0 h-8 w-full border-t border-b border-dashed border-pink-400/50"></span>
        <span className="absolute top-0 h-4 w-full border-b border-dashed border-pink-400/50"></span>
      </span>
      <span className="relative z-10 px-4 font-medium text-gray-700 dark:text-gray-200">
        Быстрая коричневая лиса перепрыгивает через ленивую собаку.
      </span>
    </div>
  }
</Example>

```html
<!-- [!code classes:align-baseline] -->
<span class="inline-block align-baseline">The quick brown fox...</span>
```

</Figure>

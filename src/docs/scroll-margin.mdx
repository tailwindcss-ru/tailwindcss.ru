### Использование отрицательных значений

Для использования отрицательного значения отступа прокрутки добавьте дефис перед именем класса, чтобы преобразовать его в отрицательное значение:

```html
<!-- [!code classes:-scroll-ml-6] -->
<div class="snap-start -scroll-ml-6 ...">
  <!-- ... -->
</div>
```

### Использование логических свойств

Используйте утилиты `scroll-ms-<number>` и `scroll-me-<number>` для установки [логических свойств](https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_Logical_Properties/Basic_concepts) `scroll-margin-inline-start` и `scroll-margin-inline-end`, которые отображаются на левую или правую сторону в зависимости от направления текста:

## Примеры

### Использование плавной прокрутки

Используйте утилиту `scroll-smooth` для включения плавной прокрутки внутри элемента:

```html
<!-- [!code classes:scroll-smooth] -->
<html class="scroll-smooth">
  <!-- ... -->
</html>
```

Установка `scroll-behavior` влияет только на события прокрутки, которые инициируются браузером.

### Использование обычной прокрутки

Используйте утилиту `scroll-auto` для возврата к поведению браузера по умолчанию для прокрутки:

```html
<!-- [!code classes:scroll-auto] -->
<html class="scroll-smooth md:scroll-auto">
  <!-- ... -->
</html>
```

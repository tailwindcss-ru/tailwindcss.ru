## Примеры

### Базовый пример

Используйте утилиту `bg-repeat` для повторения фонового изображения как по вертикали, так и по горизонтали:

<Figure>

<Example padding={false}>
  {<div className="h-64 bg-[url('/background-pattern.svg')] bg-center bg-repeat"></div>}
</Example>

```html
<!-- [!code classes:bg-repeat] -->
<div class="bg-[url(/img/clouds.svg)] bg-center bg-repeat ..."></div>
```

</Figure>

### Повторение по горизонтали

Используйте утилиту `bg-repeat-x` для повторения фонового изображения только по горизонтали:

<Figure>

<Example padding={false}>
  {<div className="h-64 bg-[url('/background-pattern.svg')] bg-center bg-repeat-x"></div>}

```html

```

</Figure>

### Повторение по вертикали

Используйте утилиту `bg-repeat-y` для повторения фонового изображения только по вертикали:

<Figure>

<Example padding={false}>
  {<div className="h-64 bg-[url('/background-pattern.svg')] bg-center bg-repeat-y"></div>}

<Example padding={false} resizable>

Используйте утилиту `bg-repeat-round` для повторения фонового изображения без обрезки, при необходимости растягивая его, чтобы избежать промежутков:

<Figure>

<Example padding={false} resizable>

</Example>

```html

```

</Figure>

### Использование CSS переменной

Используйте синтаксис <code>list-image-{'(<custom-property>)'}</code> для управления изображением маркера для элементов списка с помощью CSS переменной:

```html
<!-- [!code classes:list-image-(--my-list-image)] -->
<ul class="list-image-(--my-list-image)">
  <!-- ... -->
</ul>
```

Это просто сокращение для <code>list-image-{'[var(<custom-property>)]'}</code>, которое автоматически добавляет функцию `var()` для вас.

### Удаление изображения маркера

Используйте утилиту `list-image-none` для удаления существующего изображения маркера из элементов списка:

```html
<!-- [!code classes:list-image-none] -->
<ul class="list-image-none">
  <!-- ... -->
</ul>
```

Инвертирование

Tailwind CSS version
v2.1+

Утилиты для применения инвертированных фильтров к элементу.

Справочник классов по умолчанию

Класс
Свойства
invert-0--tw-invert: invert(0);
invert--tw-invert: invert(1);

Применение

Используйте утилиты invert и invert-0 вместе с утилитой filter, чтобы определить, должен ли элемент отображаться с инвертированными цветами или в обычном режиме.

<div class="filter invert ...">
  <!-- ... -->
</div>

Адаптивность

Чтобы управлять фильтром инвертирования элемента в определенной контрольной точке, добавьте префикс {screen}: к любой существующей утилите инвертирования. Например, используйте md:invert-0, чтобы применить утилиту invert-0 только при средних размерах экрана и выше.

<div class="filter invert md:invert-0 ...">
  <!-- ... -->
</div>

Для получения дополнительной информации о функциях адаптивного дизайна Tailwind ознакомьтесь с документацией Адаптивного дизайна.

Кастомизация

Вы можете настроить, какие утилиты invert будут генерироваться, используя ключ invert в разделе theme вашего файла tailwind.config.js.

  // tailwind.config.js
  module.exports = {
    theme: {
+     extend: {
+       invert: {
+         25: '.25',
+         50: '.5',
+         75: '.75',
+       }
+     }
    }
  }

Дополнительные сведения о настройке темы по умолчанию смотрите в документации по настройке темы.

Варианты

По умолчанию, только responsive варианты создаются для утилит invert.

Вы можете контролировать, какие варианты создаются для утилит invert для изменения свойства invert в разделе variants Вашего файла конфигурации tailwind.config.js.

Например, эта конфигурация также будет генерировать варианты hover и focus:

  // tailwind.config.js
  module.exports = {
    variants: {
      extend: {
        // ...
+       invert: ['hover', 'focus'],
      }
    }
  }

Отключение

Если Вы не планируете использовать в своем проекте утилиты для изменения invert, Вы можете полностью отключить их, установив для invert свойство значение false в разделе corePlugins Вашего файла конфигурации:

  // tailwind.config.js
  module.exports = {
    corePlugins: {
      // ...
+     invert: false,
    }
  }
